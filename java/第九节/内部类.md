# 内部类

定义在一个类中的类。

```java
class A {
    class B {
    }
}
```

`A` 为外部类，`B` 为内部类。

可分为：

- 成员内部类（静态 / 非静态）
- 局部内部类
- 匿名内部类

## 内部类特点

- 内部类可以定义属性、方法、构造等
- 静态内部类可以被 `final` 或 `abstract` 修饰
- 类被 `final` 修饰后不能被继承；被 `abstract` 修饰后不能直接 `new`
- 静态内部类不能直接访问外部类的非静态成员
- 非静态内部类的创建方式示例：`外部类.内部类 对象名 = new 外部类().new 内部类();`
- 静态内部类创建方式示例：`外部类.内部类 对象名 = new 外部类.内部类();`

## 外部类与内部类成员重名

```java
System.out.println(name);              // 内部类局部变量
System.out.println(this.name);         // 内部类成员变量
System.out.println(Outer.this.name);   // 外部类成员变量
```

## 匿名内部类

可以理解为没有显式类名的内部类，常用于临时实现接口或抽象类。

```java
public interface B {
    void method();
}

public static void main(String[] args) {
    // 把匿名实现作为参数传入
    use(new B() {
        @Override
        public void method() {
            System.out.println("匿名内部类实现");
        }
    });

    // 或先拿到接口引用再调用
    B b = create();
    b.method();
}

public static void use(B b) {
    b.method();
}

public static B create() {
    return new B() {
        @Override
        public void method() {
            System.out.println("工厂方法返回匿名实现");
        }
    };
}
```
