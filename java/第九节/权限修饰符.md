# 权限修饰符

Java 提供四种访问权限，使用不同修饰符时，被修饰成员的可见范围不同：

| 修饰符 | 含义 |
|--------|------|
| public | 公共，任何地方都能访问 |
| protected | 受保护，同包或子类可访问 |
| default（包访问） | 默认权限，不写修饰符；仅同包可访问（不能写关键字 `default`） |
| private | 私有，只能在本类中直接访问 |

| 使用范围 | public | protected | default | private |
|----------|--------|-----------|---------|---------|
| 同类 | Y | Y | Y | Y |
| 同包不同类 | Y | Y | Y | N |
| 不同包子类 | Y | Y | N | N |
| 不同包非子类 | Y | N | N | N |

权限从大到小：`public > protected > default > private`

常见习惯：

- 属性：多用 `private`
- 成员方法：多用 `public`，便于调用
- 构造方法：多用 `public`，便于 `new`
