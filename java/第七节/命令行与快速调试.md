# 命令行与快速调试

## 命令行传入参数

打开 Run/Debug Configurations，在 Program arguments（或 Build and run 相关参数栏）中输入参数即可，对应 `main(String[] args)`。

## 快速生成 / 抽取方法

1. 快捷键 `Alt + Enter`：根据提示生成或修正代码
2. 选中一段代码后按 `Ctrl + Alt + M`：抽取为方法

## Debug 调试（常用）

1. 在代码行号左侧单击，设置断点
2. 以 Debug 模式运行程序
3. 常用操作：
   - Step Over（F8）：单步跳过，进入下一行
   - Step Into（F7）：进入方法内部
   - Step Out（Shift + F8）：跳出当前方法
   - Resume（F9）：继续运行到下一个断点
4. 可在 Variables / Watches 中查看变量当前值
