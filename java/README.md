## 文件约定

- **`.md`**：概念笔记，适合速查与复习
- **`.java`**：对应知识点的示例代码，可在 IDE 中直接打开运行（部分文件为演示片段，需按注释理解）

部分早期示例文件名沿用小写开头（如 `methods.java`），属于学习笔记习惯写法；正式项目中类名与文件名建议遵循首字母大写规范。
