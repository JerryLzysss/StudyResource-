# DOS 命令

1. 打开 CMD：`Win + R`，输入 `cmd`

2. 常用指令对照：

| 作用 | 示例 |
|------|------|
| 切换盘符 | `D:` |
| 查看当前目录内容 | `dir` |
| 进入文件夹 | `cd 文件夹名` |
| 返回上一级 | `cd ..` |
| 清屏 | `cls` |
| 退到当前盘根目录 | `cd \` |
| 创建文件夹 | `mkdir 文件夹名` |
| 创建多级目录 | `mkdir a\b\c` |
| 退出窗口 | `exit` |
| 删除空文件夹 | `rd 文件夹名` |
| 删除文件 | `del 文件名` |
| 按通配符删除 | `del *.png` |
