# 关键字

## 01-关键字分类

Java 关键字约 50 个（含保留字 `goto`、`const`；具体数量随版本略有差异）：

abstract、assert、boolean、break、byte、case、catch、char、class、continue、default、do、double、else、enum、extends、final、finally、float、for、if、implements、import、int、interface、instanceof、long、native、new、package、private、protected、public、return、short、static、strictfp、super、switch、synchronized、this、throw、throws、transient、try、void、volatile、while。

2 个保留字（不能当标识符用，但目前无实际语义）：

goto、const

3 个特殊字面量（不是关键字）：

true、false、null

## 02-关键字大致含义

| 关键字 | 含义 |
|--------|------|
| abstract | 表明类或者成员方法具有抽象属性 |
| assert | 断言，用于程序调试 |
| boolean | 基本数据类型之一，布尔类型 |
| break | 提前跳出一个块 |
| byte | 基本数据类型之一，字节类型 |
| case | 用在 switch 语句中，表示其中一个分支 |
| catch | 用在异常处理中，用来捕获异常 |
| char | 基本数据类型之一，字符类型 |
| class | 声明一个类 |
| const | 保留关键字，没有具体含义 |
| continue | 结束本次循环，进入下一次循环 |
| default | 默认分支，例如用在 switch 中 |
| do | 用在 do-while 循环结构中 |
| double | 基本数据类型之一，双精度浮点数 |
| else | 条件不成立时的分支 |
| enum | 枚举 |
| extends | 继承关系（类或接口） |
| final | 最终的：可修饰类、方法、变量 |
| finally | 异常处理中几乎总会执行的代码块 |
| float | 基本数据类型之一，单精度浮点数 |
| for | 循环结构的引导词 |
| goto | 保留关键字，没有具体含义 |
| if | 条件语句的引导词 |
| implements | 类实现接口 |
| import | 导入指定的类或包 |
| instanceof | 判断对象是否为某类型的实例 |
| int | 基本数据类型之一，整数类型 |
| interface | 接口 |
| long | 基本数据类型之一，长整数类型 |
| native | 声明方法由本地语言（如 C/C++）实现 |
| new | 创建新实例对象 |
| package | 包 |
| private | 访问控制：私有 |
| protected | 访问控制：受保护 |
| public | 访问控制：公有 |
| return | 从方法中返回 |
| short | 基本数据类型之一，短整数类型 |
| static | 静态属性 |
| strictfp | 声明浮点运算遵循 IEEE 754 规范 |
| super | 父类引用或父类构造方法 |
| switch | 分支语句结构 |
| synchronized | 同步执行 |
| this | 当前实例对象的引用 |
| throw | 抛出一个异常 |
| throws | 声明方法可能抛出的异常 |
| transient | 声明该字段不参与序列化 |
| try | 尝试执行可能抛出异常的代码块 |
| void | 方法没有返回值 |
| volatile | 保证多线程可见性相关语义 |
| while | 循环结构 |
| null | 字面量，表示引用类型的空值（不是关键字） |
