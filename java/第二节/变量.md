# 变量

1. 概述：在程序运行过程中，值可以改变的数据
2. 分类：基本数据类型与引用数据类型

| 类型 | 含义 |
|------|------|
| **基本数据类型** | |
| int | 整型，4 个字节 |
| byte | 字节型，1 个字节 |
| short | 短整型，2 个字节 |
| long | 长整型，8 个字节 |
| float | 单精度浮点，4 个字节 |
| double | 双精度浮点，8 个字节 |
| char | 字符型，2 个字节 |
| boolean | 布尔类型（JVM 规范未严格规定字节数，常用说法为 1 字节） |
| **引用数据类型** | |
| String 等 | 不属于基本类型，属于引用类型 |
