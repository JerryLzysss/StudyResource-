# 类型转换

1. 类型不一致时需要考虑转换
2. 不同类型数据做运算时会发生类型提升
3. 自动类型转换（小 → 大）
4. 强制类型转换（大 → 小）

转换方向大致为：

`byte, short, char → int → long → float → double`

5. 精度损失与数据溢出

```java
int i = (int) 2.9;           // float/double => int，小数部分丢失
int j = (int) 10000000000000L; // long => int，超出 int 范围会溢出截断
```

6. `byte`、`short` 定义时，若等号右边是整数常量且不超过该类型范围，则不必手动强制转换
7. `char` 参与运算时会自动提升为 `int`（按 Unicode/ASCII 码值计算）
8. 整数溢出时按补码规则回绕，结果可能不符合直觉，应避免溢出
