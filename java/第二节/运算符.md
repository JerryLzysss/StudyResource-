# 运算符

## 算术运算

`+`、`-`、`*`、`/`、`%`（取模）

## 赋值运算

`=`、`+=`、`-=`、`*=`、`/=` 等

## 关系（比较）运算

`==`、`>`、`<`、`>=`、`<=`、`!=`

## 逻辑运算

`&&`（与）、`||`（或）、`!`（非）

## 短路逻辑

当左侧结果已经能确定整个表达式结果时，**不再计算**右侧表达式（与循环无关）。

例如：`a >= 0 && b >= 0`，若 `a >= 0` 为 false，则不会再判断 `b >= 0`。  
同理：`a > b || c > d`，若 `a > b` 为 true，则不会再判断 `c > d`。

## 三元运算符

格式：`条件 ? 表达式1 : 表达式2`

例如：`a > b ? 1 : 2`  
含义：若 `a > b` 为真则结果为 1，否则为 2。
