# 进制转换

## 10 → 2（不断除以 2 取余）

```
100/2 => 50 余 0
50/2  => 25 余 0
25/2  => 12 余 1
12/2  => 6  余 0
6/2   => 3  余 0
3/2   => 1  余 1
1/2   => 0  余 1
```

余数从下往上读：`100(10) = 1100100(2)`

## 2 → 10（按位加权求和）

从右到左，第 n 位（从 0 起）乘以 `2^n` 再相加：

```
1100100
= 2^0*0 + 2^1*0 + 2^2*1 + 2^3*0 + 2^4*0 + 2^5*1 + 2^6*1
= 0 + 0 + 4 + 0 + 0 + 32 + 64
= 100
```

## 8 → 2（每位八进制转 3 位二进制）

`100(8) => 001 000 000`

## 2 → 8（每 3 位二进制转 1 位八进制）

`111101000 => 111 101 000 => 750(8)`

## 16 → 2（每位十六进制转 4 位二进制）

`158A(16) => 0001 0101 1000 1010`

## 2 → 16（每 4 位二进制转 1 位十六进制）

`111101001010 => 1111 0100 1010 => F4A`
