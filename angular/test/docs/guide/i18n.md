# 国际化 (i18n)

Angular 的国际化特性处理消息翻译和格式化，包括对 Unicode 标准 ICU 语法的支持。

## 标记需要翻译的文本

在模板中使用 `i18n` 属性：

```html
<h1 i18n="@@welcomeMessage">欢迎使用 Angular</h1>
<p i18n>你好，{{ name }}！</p>
```

## 提取翻译消息

```bash
ng extract-i18n --output-path src/locale
```

这会在 `src/locale` 目录生成 `messages.xlf` 文件。

## 构建本地化版本

```bash
ng build --localize
```

为每种语言生成独立的构建产物。

## ICU 表达式

支持复数、性别等复杂格式：

```html
<span i18n>
  {count, plural, =0 {没有项目} =1 {1 个项目} other {{{count}} 个项目}}
</span>
```

## 运行时语言切换

对于需要运行时切换语言的应用，可以使用 `@angular/localize` 或第三方库如 `ngx-translate`。

## 日期和数字格式化

```typescript
import { registerLocaleData } from '@angular/common';
import localeZh from '@angular/common/locales/zh';

registerLocaleData(localeZh);
```

在 `app.config.ts` 中设置 locale：

```typescript
providers: [{ provide: LOCALE_ID, useValue: 'zh-CN' }]
```
