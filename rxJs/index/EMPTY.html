RxJS EMPTY

核心概念：EMPTY 是一个常量 Observable，不发出任何 next 值，订阅后立即 complete。应使用 EMPTY 常量，而非已弃用的 empty() 函数。

声明

const EMPTY: Observable<never>;

描述

任何不带 scheduler 参数的 empty() 调用返回的相同 Observable 实例。最好使用 EMPTY 常量。

EMPTY 创建的 Observable 不向观察者发出任何项目，并立即发出 complete 通知。常用于与其他 Observable 组合，例如在 mergeMap 中跳过某些分支。

例子

与 startWith 组合

import { EMPTY } from 'rxjs'; import { startWith } from 'rxjs/operators'; const result = EMPTY.pipe(startWith(7)); result.subscribe(x => console.log(x)); // 输出: 7，然后 complete

在 mergeMap 中跳过偶数

import { EMPTY, interval, of } from 'rxjs'; import { mergeMap, take } from 'rxjs/operators'; const result = interval(1000).pipe( take(5), mergeMap(x => x % 2 === 1 ? of('a', 'b', 'c') : EMPTY) ); result.subscribe(x => console.log(x));
