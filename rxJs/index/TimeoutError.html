RxJS TimeoutError

核心概念：TimeoutError 是超时时间到期后引发的错误，通常由 timeout 运算符在指定时间内未收到发射时抛出。

接口定义

interface TimeoutError extends Error { }

描述

当 Observable 在 timeout 运算符设定的时限内没有发出任何值时，会抛出 TimeoutError。该错误继承自 JavaScript 标准 Error 对象，可通过 Observable 的 error 回调捕获。

例子

import { timer } from 'rxjs'; import { timeout } from 'rxjs/operators'; // 要求 1 秒内必须收到值，但 timer(3000) 3 秒后才发出 timer(3000).pipe( timeout(1000) ).subscribe({ next: v => console.log(v), error: err => console.error(err.name) // TimeoutError });
