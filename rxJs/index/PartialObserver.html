RxJS PartialObserver

核心概念：PartialObserver<T> 是局部观察者的联合类型，允许只提供 next、error、complete 中的部分回调。

类型定义

type PartialObserver<T> = | NextObserver<T> | ErrorObserver<T> | CompletionObserver<T>;

组成类型

类型必需回调说明
NextObserver<T>next必须提供 next，error/complete 可选
ErrorObserver<T>error必须提供 error，next/complete 可选
CompletionObserver<T>complete必须提供 complete，next/error 可选

描述

与完整 Observer（三个回调均必需）不同，PartialObserver 允许按需只实现关心的通知类型。未提供的回调对应的通知会被忽略。

Observable.subscribe() 接受 PartialObserver<T> 作为参数，也支持将 next、error、complete 作为三个独立函数参数传入。

例子

import { of, throwError } from 'rxjs'; // NextObserver：只关心 next of(1, 2, 3).subscribe({ next: x => console.log(x) }); // ErrorObserver：只关心 error throwError(() => new Error('fail')).subscribe({ error: err => console.error(err.message) }); // CompletionObserver：只关心 complete of('done').subscribe({ complete: () => console.log('完成') });
