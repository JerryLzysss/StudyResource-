RxJS ErrorObserver

核心概念：ErrorObserver 是 PartialObserver 的一种，必须提供 error 回调，用于处理 Observable 的错误通知。

接口

interface ErrorObserver<T> { closed?: boolean next?: (value: T) => void error: (err: any) => void complete?: () => void }

属性

属性类型
closedboolean
next(value: T) => void（可选）
error(err: any) => void（必需）
complete() => void（可选）

描述

ErrorObserver 与 NextObserver、CompletionObserver 共同组成 PartialObserver<T> 联合类型。订阅 Observable 时，可以传入只包含 error 回调的 ErrorObserver，此时 next 和 complete 通知会被忽略（若未提供对应回调）。

例子

import { throwError } from 'rxjs'; const errorObserver = { error: err => console.error('捕获错误:', err.message) }; throwError(() => new Error('出错了')).subscribe(errorObserver);
