RxJS ArgumentOutOfRangeError

核心概念：当在 Observable 的某个索引处查询元素，但该序列中不存在这样的索引或位置时，会抛出 ArgumentOutOfRangeError 错误。

类型定义

interface ArgumentOutOfRangeError extends Error { }

说明

ArgumentOutOfRangeError 是 RxJS 内置的错误类型，通常在以下场景中出现：

  • 使用 elementAt 访问超出 Observable 序列范围的索引
  • 使用 take 或 takeLast 等运算符时传入负数或无效参数

该错误继承自 JavaScript 标准 Error 对象，可通过 Observable 的 error 回调捕获。

代码示例

elementAt 索引越界

import { of } from 'rxjs'; import { elementAt } from 'rxjs/operators'; of(1, 2, 3).pipe( elementAt(5) // 索引 5 不存在，抛出 ArgumentOutOfRangeError ).subscribe({ next: x => console.log(x), error: err => console.error(err.name, err.message) });
