RxJS config

核心概念：config 是 RxJS 的全局配置对象，用于配置诸如创建 Promise 时使用的 Promise 构造函数等全局行为。

声明

const config: { Promise: PromiseConstructorLike; onUnhandledError: ((err: any) => void) | null; useDeprecatedSynchronousErrorHandling: boolean; };

描述

RxJS 的全局配置对象，用于配置诸如 Promise 构造函数应用于创建 Promise 的内容，以及未捕获错误的处理方式。

常用配置项

属性说明
Promise指定 Observable 转 Promise 时使用的 Promise 构造函数（如 firstValueFrom、lastValueFrom）
onUnhandledError全局未处理错误回调，默认为 null
useDeprecatedSynchronousErrorHandling是否使用已弃用的同步错误处理模式

代码示例

import { config } from 'rxjs'; // 查看当前 Promise 构造函数 console.log(config.Promise); // 设置全局未处理错误处理器 config.onUnhandledError = (err) => { console.error('未处理的 RxJS 错误:', err); };
