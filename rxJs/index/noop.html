RxJS noop

核心概念：noop 是一个空函数，不执行任何操作。RxJS 内部在 NEVER 等 Observable 的订阅逻辑中使用它。

签名

function noop(): void

参量

没有参数。

返回值

void — 无返回值。

描述

noop 是最简单的工具函数：调用它不会产生任何副作用。在 RxJS 源码中，NEVER 的实现为 new Observable(noop)，订阅时执行 noop 即不做任何事。

Notification 的 observe / do 方法在缺少对应回调时也会静默 noop，不会抛出错误。

例子

import { noop } from 'rxjs'; noop(); // 无效果 noop(undefined); // 忽略多余参数（函数本身不接收参数） // NEVER 内部实现示意 // export const NEVER = new Observable(noop);
