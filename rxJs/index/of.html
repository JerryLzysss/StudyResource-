RxJS of

核心概念：of 是创建函数，将参数转换为 Observable，按顺序同步发出每个参数作为 next 通知，然后完成。

签名

of<T>(...values: T[]): Observable<T>

参量

  • values：要依次发出的值，可以是任意数量、任意类型

返回值

Observable：同步发出所有参数后立刻 complete 的 Observable。

描述

与 from 不同，of 不会对数组或可迭代对象做展开，而是将每个参数整体作为一个 next 值发出。例如 of([1, 2, 3]) 只会发出一个元素 [1, 2, 3]。

适合在需要 Observable 的地方直接返回固定值，或在 mergeMap 等组合运算符中包装同步结果。

例子

发出 10、20、30

import { of } from 'rxjs'; of(10, 20, 30).subscribe({ next: value => console.log('next:', value), error: err => console.log('error:', err), complete: () => console.log('结束') }); // 输出: // next: 10 // next: 20 // next: 30 // 结束

发出数组整体（不展开）

import { of } from 'rxjs'; of([1, 2, 3]).subscribe({ next: value => console.log('next:', value), complete: () => console.log('结束') }); // 输出: // next: [1, 2, 3] // 结束
