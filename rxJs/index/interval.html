RxJS interval

核心概念：interval 创建一个 Observable，按指定的时间间隔发出递增的整数序列（0, 1, 2, …），常用于定时任务和轮询。

签名

interval(period: number = 0, scheduler?: SchedulerLike): Observable<number>

参量

  • period（可选，默认 0）：两次发射之间的时间间隔，单位为毫秒
  • scheduler（可选）：用于调度值发射的 SchedulerLike，默认使用 async 调度器

描述

interval 返回一个 Observable，它发出无限的升序整数序列，发射之间具有您选择的固定时间间隔 period。第一次发射不会立即发生，而是在第一个 period 过去之后才发出 0。

默认情况下，该函数使用 async SchedulerLike 来提供时间概念，但您可以将任何 SchedulerLike 传递给它。与 timer 不同，interval 无法指定首次发射的延迟时间；若需立即开始发射，可使用 timer(0, period)。

由于 interval 发出无限序列，通常需要配合 take 或手动 unsubscribe 来停止。

例子

每秒发出升序数字，最多 4 个

import { interval, take } from 'rxjs'; const numbers = interval(1000); const takeFourNumbers = numbers.pipe(take(4)); takeFourNumbers.subscribe(x => console.log('下一个: ', x)); // 输出： // Next: 0 // Next: 1 // Next: 2 // Next: 3

订阅与取消

import { interval } from 'rxjs'; const observable = interval(1000); const subscription = observable.subscribe(x => console.log(x)); // 稍后取消： subscription.unsubscribe();
