RxJS UnsubscriptionError

核心概念：UnsubscriptionError 是在 Subscription 的 unsubscribe() 运行期间发生一个或多个错误时抛出的错误。

接口定义

interface UnsubscriptionError extends Error { readonly errors: any[] }

属性

属性类型描述
errorsany[]只读。取消订阅过程中收集到的所有错误

描述

当 Subscription 执行 unsubscribe() 时，会依次调用所有注册的拆解逻辑（teardown）。如果其中某个拆解函数抛出异常，RxJS 会收集这些错误并最终抛出 UnsubscriptionError，其中 errors 数组包含所有捕获到的异常。

例子

import { Subscription } from 'rxjs'; const sub = new Subscription(); // 添加会在 unsubscribe 时抛出错误的拆解逻辑 sub.add(() => { throw new Error('拆解失败 1'); }); sub.add(() => { throw new Error('拆解失败 2'); }); try { sub.unsubscribe(); } catch (err) { console.error(err.name); // UnsubscriptionError console.error(err.errors); // [Error, Error] }
