RxJS VirtualTimeScheduler

核心概念：VirtualTimeScheduler 是虚拟时间调度器，用于测试场景，可以手动推进"虚拟时间"而无需等待真实时间流逝。

类定义

class VirtualTimeScheduler extends AsyncScheduler { protected static frameTimeFactor: number constructor( SchedulerAction: typeof AsyncAction = VirtualAction, maxFrames: number = Number.POSITIVE_INFINITY ) frame: number index: number maxFrames: number flush(): void }

构造函数

constructor(SchedulerAction?, maxFrames?)

  • SchedulerAction（可选，默认 VirtualAction）：调度动作类型
  • maxFrames（可选，默认 Infinity）：最大帧数

属性

属性类型描述
framenumber当前虚拟时间帧
indexnumber当前索引
maxFramesnumber最大帧数限制

方法

flush(): void — 提示调度程序执行其所有排队的操作，从而清理队列。

描述

VirtualTimeScheduler 主要用于 RxJS 测试（如 TestScheduler、大理石测试）。通过调用 flush() 可以立即执行所有已调度的任务，无需等待真实延迟，使异步代码的测试变得确定且快速。

例子

import { VirtualTimeScheduler } from 'rxjs'; const scheduler = new VirtualTimeScheduler(); scheduler.schedule(() => console.log('帧 10'), 10); scheduler.schedule(() => console.log('帧 20'), 20); console.log('flush 前，帧:', scheduler.frame); // 0 scheduler.flush(); console.log('flush 后，帧:', scheduler.frame); // 20 // 输出： frame 10, frame 20
