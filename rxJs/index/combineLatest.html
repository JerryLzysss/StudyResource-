RxJS combineLatest

核心概念：combineLatest 组合多个 Observable，创建一个 Observable，其值是根据每个输入 Observable 的最新值计算得出的。

签名

combineLatest<O extends ObservableInput<any>, R>( ...observables: (SchedulerLike | O | ((...values: ObservedValueOf<O>[]) => R))[] ): Observable<R>

描述

每当任何 Observable 输入发出一个值时，它都会使用来自所有输入的最新值来计算公式，然后发出该公式的输出。

combineLatest 合并所有作为参数传递的 Observable 的值。这是通过按顺序订阅每个 Observable 来完成的，并且每当发出任何 Observable 时，就从每个 Observable 中收集最新值的数组。因此，如果将 n 个 Observables 传递给运算符，则返回的 Observable 将始终发出 n 个值数组。

为确保输出数组的长度始终相同，combineLatest 实际上将等待所有输入 Observable 至少发出一次，然后再开始发出结果。传递空数组将导致 Observable 立即完成。

如果至少有一个 Observable 传递给了 combineLatest 并且所有 Observable 都发出了值，那么当所有组合流完成时，结果 Observable 将完成。如果有任何 Observable 出错，combineLatest 也会立即发生错误。

combineLatest 接受可选的 project 函数作为参数，该函数接受通常由结果 Observable 发出的所有值作为参数，可以返回任何类型的值。

例子

结合两个 timer Observables

import { combineLatest, timer } from 'rxjs'; const firstTimer = timer(0, 1000); const secondTimer = timer(500, 1000); const combinedTimers = combineLatest([firstTimer, secondTimer]); combinedTimers.subscribe(value => console.log(value)); // 输出: // [0, 0] 在 0.5 秒后 // [1, 0] 在 1 秒后 // [1, 1] 在 1.5 秒后 // [2, 1] 在 2 秒后

使用 project 函数动态计算 BMI

import { combineLatest, of } from 'rxjs'; import { map } from 'rxjs/operators'; const weight = of(70, 72, 76, 79, 75); const height = of(1.76, 1.77, 1.78); const bmi = combineLatest([weight, height]).pipe( map(([w, h]) => w / (h * h)) ); bmi.subscribe(x => console.log('BMI 为 ' + x)); // 输出: // BMI is 24.212293388429753 // BMI is 23.93948099205209 // BMI is 23.671253629592222
