RxJS Subscribable

核心概念：Subscribable 是可订阅对象的接口，定义了 subscribe 方法，是 Observable 体系的基础契约之一。

接口定义

interface Subscribable<T> { subscribe(observer: Partial<Observer<T>>): Unsubscribable }

描述

Subscribable 是 RxJS 中最基本的可订阅接口。任何实现了 subscribe 方法并返回 Unsubscribable 的对象都符合 Subscribable 契约。

Observable 和 Subject 都实现了 Subscribable。RxJS 的 from 等函数会检查输入是否为 Subscribable 来决定如何转换。

子接口

类实现

  • Observable

方法

方法说明
subscribe(observer)订阅并接收通知，返回 Unsubscribable 用于取消

例子

// 自定义 Subscribable 对象 const customSubscribable = { subscribe(observer) { observer.next('hello'); observer.complete(); return { unsubscribe() {} }; } }; // RxJS from 可识别 Subscribable import { from } from 'rxjs'; from(customSubscribable).subscribe(console.log); // 输出: hello
