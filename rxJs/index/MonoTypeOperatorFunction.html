RxJS MonoTypeOperatorFunction

核心概念：MonoTypeOperatorFunction 是 OperatorFunction 的特例，表示输入和输出 Observable 的类型相同（T → T），如 filter、tap 等运算符。

类型定义

interface MonoTypeOperatorFunction<T> extends OperatorFunction<T, T> { // inherited from OperatorFunction // inherited from UnaryFunction (source: Observable<T>): Observable<T> }

描述

MonoTypeOperatorFunction 描述一种函数类型：接受 Observable<T> 并返回 Observable<T>，即不改变流中值的类型。它是 OperatorFunction<T, T> 的别名。

与之相对，普通的 OperatorFunction<T, R> 允许输入类型 T 与输出类型 R 不同，例如 map 可以将 number 映射为 string。

常见的 MonoTypeOperatorFunction 包括：filter、tap、take、skip、distinctUntilChanged 等。

例子

import { of } from 'rxjs'; import { filter, tap } from 'rxjs/operators'; // filter 和 tap 都是 MonoTypeOperatorFunction<number> of(1, 2, 3, 4, 5).pipe( tap(x => console.log('tap:', x)), filter(x => x % 2 === 0) ).subscribe(x => console.log('out:', x)); // tap: 1 tap: 2 out: 2 tap: 3 tap: 4 out: 4 tap: 5
