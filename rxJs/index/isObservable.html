RxJS isObservable

核心概念：isObservable 用于判断给定对象是否为 RxJS Observable，是库推荐的类型检测方式。

签名

isObservable<T>(obj: any): obj is Observable<T>

参量

  • obj：待测试的对象

返回值

obj is Observable<T> — TypeScript 类型守卫，运行时返回 boolean。

描述

isObservable 是 RxJS 提供的官方工具函数，用于可靠地检测某个值是否为 RxJS（5+）Observable。相比 instanceof 或 typeof obj.subscribe === 'function' 等手写检测，它能更好地处理跨模块副本和兼容性问题。

例子

import { isObservable, of, interval } from 'rxjs'; console.log(isObservable(of(1))); // true console.log(isObservable(interval())); // true console.log(isObservable([1, 2, 3])); // false console.log(isObservable(null)); // false
