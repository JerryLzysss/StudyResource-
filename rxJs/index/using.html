RxJS using

核心概念：using 创建一个使用资源的 Observable，该资源将与 Observable 同时被处理，在完成、出错或取消订阅时自动清理。

签名

using( resourceFactory: () => void | Unsubscribable, observableFactory: (resource: void | Unsubscribable) => ObservableInput ): Observable

参量

参数说明
resourceFactory创建任何实现 unsubscribe 方法的资源对象的函数
observableFactory创建一个可使用注入资源对象的 Observable 的函数

描述

当您在 Observable 执行期间需要自动清理资源时，请使用 using。

using 是工厂运算符，它接受两个函数。第一个函数返回一次性资源（实现 unsubscribe 方法的对象）。第二个函数接收该资源并返回 Observable。每当有人订阅时，这两个函数都会被调用。

当 Observable 完成、出错或被取消订阅时，资源对象的 unsubscribe 方法会被自动调用，确保资源在适当的时间被处置。

例子

import { using, of } from 'rxjs'; const source = using( () => ({ value: 42, unsubscribe() { console.log('资源已清理'); } }), resource => of(`使用资源: ${resource.value}`) ); source.subscribe({ next: v => console.log(v), complete: () => console.log('完成') }); // 输出： // 使用资源: 42 // 完成 // 资源已清理
