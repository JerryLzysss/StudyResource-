RxJS Unsubscribable

核心概念：Unsubscribable 是订阅接口，定义了具有 unsubscribe() 方法的对象，用于释放资源或取消执行。

接口定义

interface Unsubscribable { unsubscribe(): void }

子接口

  • SubscriptionLike

方法

unsubscribe(): void

释放资源、取消订阅。不带任何参数。

描述

Unsubscribable 是 RxJS 中最基础的"可取消"接口。任何实现了 unsubscribe() 方法的对象都符合此接口，包括 Subscription、Subject 以及 using 运算符创建的资源对象。

例子

import { interval } from 'rxjs'; // Subscription 实现了 Unsubscribable const resource: Unsubscribable = interval(1000).subscribe(x => console.log(x)); // 稍后取消 resource.unsubscribe();
