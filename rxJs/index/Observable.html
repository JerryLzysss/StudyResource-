RxJS Observable

核心概念：Observable 表示在任何时间量内的任何一组值，是 RxJS 的最基本构建块。通过 subscribe 启动执行并注册 Observer 处理程序。

类声明

class Observable<T> implements Subscribable<T> { static create: Function static if: typeof iif static throw: typeof throwError constructor(subscribe?: (this: Observable<T>, subscriber: Subscriber<T>) => TeardownLogic) source: Observable<any> operator: Operator<any, T> lift<R>(operator: Operator<T, R>): Observable<R> subscribe(observerOrNext?, error?, complete?): Subscription pipe(...operations: OperatorFunction<any, any>[]): Observable<any> forEach(next: (value: T) => void): Promise<void> toPromise(): Promise<T> }

子类

  • Subject（含 BehaviorSubject、ReplaySubject、AsyncSubject）
  • ConnectableObservable
  • GroupedObservable

主要方法

方法说明
subscribe()调用 Observable 的执行并注册 Observer，返回 Subscription
pipe()将多个操作符串联成管道
lift()创建带新操作符的 Observable
forEach()以 Promise 方式消费 Observable

描述

subscribe 不是普通操作符，而是启动 Observable 内部执行的方法。调用 subscribe 才是 Observable 真正开始工作的时刻。

可以传入 Observer 对象，或依次传入 next、error、complete 三个回调函数。返回的 Subscription 可用于 unsubscribe() 停止执行并清理资源。

例子

订阅 Observer 对象

import { of } from 'rxjs'; const sumObserver = { sum: 0, next(value) { console.log('累加: ' + value); this.sum = this.sum + value; }, complete() { console.log('总和等于: ' + this.sum); } }; of(1, 2, 3).subscribe(sumObserver); // 累加: 1, 2, 3 → 总和等于: 6

订阅回调函数

import { of } from 'rxjs'; let sum = 0; of(1, 2, 3).subscribe( value => { sum += value; console.log('累加: ' + value); }, undefined, () => console.log('总和等于: ' + sum) );

取消订阅

import { interval } from 'rxjs'; const subscription = interval(1000).subscribe(num => console.log(num)); setTimeout(() => { subscription.unsubscribe(); console.log('已取消订阅！'); }, 2500);
