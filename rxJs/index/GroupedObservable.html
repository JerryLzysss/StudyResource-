RxJS GroupedObservable

核心概念：GroupedObservable 表示属于由公共键标识的同一组的值。它发出的值来自源 Observable，分组键可通过实例上的 key 字段访问。

类定义

class GroupedObservable<K, T> extends Observable { constructor(key: K, groupSubject: Subject<T>, refCountSubscription?: RefCountSubscription) key: K _subscribe(subscriber: Subscriber<T>) }

描述

GroupedObservable 是 Observable 的子类，通常由 groupBy 运算符创建。每个 GroupedObservable 代表源流中某个分组键对应的所有值。

公共密钥可用作 key GroupedObservable 实例上的字段，用于识别当前分组。

构造器

constructor(key: K, groupSubject: Subject, refCountSubscription?: RefCountSubscription)
  • key（类型 K）：分组键
  • groupSubject（类型 Subject）：内部 Subject，用于向该组推送值
  • refCountSubscription（可选）：引用计数订阅，用于管理组的生命周期

属性

属性类型描述
keyK在构造函数中声明的分组键

方法

_subscribe(subscriber: Subscriber) — 内部订阅方法，将订阅者连接到组 Subject。

例子

import { of } from 'rxjs'; import { groupBy, mergeMap } from 'rxjs/operators'; of( { id: 1, name: 'JavaScript' }, { id: 2, name: 'TypeScript' }, { id: 1, name: 'RxJS' } ).pipe( groupBy(p => p.id), mergeMap(group$ => group$.pipe( mergeMap(val => of(`key=${group$.key}: ${val.name}`)) ) ) ).subscribe(console.log);
