RxJS Subscription

核心概念：Subscription 表示一次性资源，例如 Observable 的执行。它具有最重要的方法 unsubscribe()，用于处置该订阅所拥有的资源。

类定义

class Subscription implements SubscriptionLike { static EMPTY: Subscription constructor(unsubscribe?: () => void) closed: boolean unsubscribe(): void add(teardown: TeardownLogic): Subscription remove(subscription: Subscription): void }

子类

  • Subscriber

描述

此外，可以通过 add() 方法将订阅分组在一起，该方法会将子订阅附加到当前订阅。取消订阅后，其所有子项（及其子孙）也将被取消订阅。

静态属性

属性类型描述
EMPTYSubscription空的、已关闭的 Subscription 实例

方法

方法说明
unsubscribe()处置订阅所拥有的资源，取消 Observable 执行
add(teardown)添加在 unsubscribe 期间要调用的拆解逻辑，也可用于添加子订阅
remove(subscription)从内部订阅列表中移除指定的子订阅

例子

基本订阅与取消

import { interval } from 'rxjs'; const observable = interval(1000); const subscription = observable.subscribe(x => console.log(x)); // 5 秒后取消订阅 setTimeout(() => subscription.unsubscribe(), 5000);

组合订阅

import { interval } from 'rxjs'; const sub1 = interval(400).subscribe(x => console.log('第一个: ' + x)); const sub2 = interval(300).subscribe(x => console.log('第二个: ' + x)); sub1.add(sub2); // 取消 sub1 会同时取消 sub2 sub1.unsubscribe();
