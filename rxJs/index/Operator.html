RxJS Operator

核心概念：Operator 是 RxJS 内部接口，描述旧式链式运算符如何通过 call 方法订阅源 Observable 并连接 Subscriber。属于内部实现细节，请勿直接使用。

接口

interface Operator<T, R> { call(subscriber: Subscriber<R>, source: any): TeardownLogic }

弃用说明

内部实现细节，请勿直接使用。在 v8 中将变为 internal。

方法

call()

参数类型
subscriberSubscriber<R>
sourceany

returns TeardownLogic

描述

在 RxJS 5 及更早版本中，运算符以 Observable.prototype.map 等形式挂载在原型上，内部通过 Operator 接口的 call 方法执行。

自 RxJS 5.5 起，推荐使用 pipeable 运算符（返回 OperatorFunction），通过 observable.pipe(map(...)) 组合，而非直接依赖 Operator 接口。

Observable.lift(operator) 内部仍使用 Operator 将运算符应用到源 Observable。
