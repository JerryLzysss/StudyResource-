RxJS forkJoin

核心概念：forkJoin 等待所有输入 Observable 完成，然后发出每个 Observable 的最后一个值，组成数组或字典。类似 Promise.all。

签名

forkJoin(...args: any[]): Observable<any>

描述

等待 Observables 完成，然后合并它们发出的最后值；若传入空数组则立即 complete。

forkJoin 接受数组或字典形式的输入 Observable。所有输入都 emit 并 complete 后，才 emit 一次结果（数组或对象），然后 complete。若任一输入 error，forkJoin 立即 error 并取消其他订阅。

注意：所有输入 Observable 必须 complete，否则 forkJoin 永远不会 emit。

例子

使用字典形式的 observable 输入

import { forkJoin, of, timer } from 'rxjs'; const observable = forkJoin({ foo: of(1, 2, 3, 4), bar: Promise.resolve(8), baz: timer(4000) }); observable.subscribe({ next: value => console.log(value), complete: () => console.log('这就是结束！') }); // 输出： // { foo: 4, bar: 8, baz: 0 } after 4 seconds // '这就是结束！' 紧随其后

使用数组形式的 observable 输入

import { forkJoin, of, timer } from 'rxjs'; const observable = forkJoin([ of(1, 2, 3, 4), Promise.resolve(8), timer(4000) ]); observable.subscribe({ next: value => console.log(value), complete: () => console.log('这就是结束！') }); // 输出： // [4, 8, 0] after 4 seconds // '这就是结束！' 紧随其后
