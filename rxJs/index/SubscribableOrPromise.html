RxJS SubscribableOrPromise

已弃用：请勿使用此类型。大多数情况下应使用 ObservableInput。将在 v8 中移除。
核心概念：SubscribableOrPromise 是联合类型，表示可订阅对象或 Promise 的输入源。

类型定义

type SubscribableOrPromise<T> = | Subscribable<T> | Subscribable<never> | PromiseLike<T> | InteropObservable<T>;

描述

SubscribableOrPromise 是 RxJS 早期版本中用于描述 from 等函数可接受输入类型的联合类型。它涵盖了：

  • Subscribable<T> — 具有 subscribe 方法的对象
  • PromiseLike<T> — 类 Promise 对象
  • InteropObservable<T> — 实现 Symbol.observable 的对象

在 RxJS 7+ 中，推荐使用更完整的 ObservableInput<T> 类型，它还包括 Array、Iterable、AsyncIterable 等更多输入形式。

例子

import { from } from 'rxjs'; // Subscribable 形式 from({ subscribe: obs => { obs.next(1); obs.complete(); return {}; } }); // Promise 形式 from(Promise.resolve('done')); // InteropObservable 形式 from({ [Symbol.observable]: () => ({ subscribe: ... }) });
