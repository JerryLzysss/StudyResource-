RxJS pipe

核心概念：pipe 是工具函数，用于组合多个 UnaryFunction；也可通过 Observable.prototype.pipe 将管道运算符串联到 Observable 上。

签名

pipe(...fns: UnaryFunction<any, any>[]): UnaryFunction<any, any>

描述

pipe 可接受一个或多个函数，每个函数接受一个参数并返回值。它返回一个新函数：将初始参数传给第一个函数，结果传给第二个，依此类推。

RxJS 中有两种用法：

  • Observable.pipe(...)：实例方法，将 pipeable 运算符组合到当前 Observable
  • import { pipe } from 'rxjs'：工具函数，从多个运算符构建可复用的组合函数

可管道运算符（pipeable operator）返回签名为 <T, R>(source: Observable<T>) => Observable<R> 的函数，即 OperatorFunction。

例子

Observable.pipe 组合运算符

import { of } from 'rxjs'; import { filter, map, scan } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( filter(x => x % 2 === 0), map(x => x + x), scan((acc, x) => acc + x, 0) ).subscribe(x => console.log(x)); // 4, 8, 12

pipe 工具函数构建可复用组合

import { pipe } from 'rxjs'; import { filter, map } from 'rxjs/operators'; const oddSquare = pipe( filter((n: number) => n % 2 === 1), map(n => n * n) ); // oddSquare 可复用于任意 Observable import { of } from 'rxjs'; of(1, 2, 3, 4, 5).pipe(oddSquare).subscribe(console.log); // 1, 9, 25
