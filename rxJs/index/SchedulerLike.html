RxJS SchedulerLike

核心概念：SchedulerLike 是调度程序的接口类型，定义了如何安排任务执行以及获取当前时间戳。

接口定义

interface SchedulerLike extends TimestampProvider { schedule<T>( work: (this: SchedulerAction<T>, state?: T) => void, delay?: number, state?: T ): Subscription now(): number }

描述

SchedulerLike 是 RxJS 调度系统的核心接口。任何实现了 schedule 和 now 方法的对象都可以作为调度程序使用。

RxJS 内置多种 SchedulerLike 实现：

  • asyncScheduler（async）— 类似 setTimeout，用于延迟和间隔任务
  • asapScheduler（asap）— 尽快执行，类似 setTimeout(fn, 0)
  • queueScheduler（queue）— 在当前调用栈中递归执行
  • animationFrameScheduler（animationFrame）— 使用 requestAnimationFrame

方法

方法说明
schedule(work, delay?, state?)安排 work 函数在 delay 毫秒后执行，返回 Subscription
now()返回当前时间戳（继承自 TimestampProvider）

类实现

  • VirtualTimeScheduler

例子

import { asyncScheduler, interval } from 'rxjs'; // 使用 asyncScheduler 延迟任务 asyncScheduler.schedule(() => console.log('delayed'), 2000); // interval 默认使用 asyncScheduler interval(1000).subscribe(v => console.log(v));
