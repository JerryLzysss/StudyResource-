RxJS ObservedValueOf

核心概念：ObservedValueOf 是 TypeScript 条件类型，从 ObservableInput 中提取其发出的值类型 T。

类型定义

type ObservedValueOf<O> = O extends ObservableInput<infer T> ? T : never;

描述

ObservedValueOf 是编译时类型工具，运行时不存在。当函数接受 ObservableInput 并需要推断内部值类型时，RxJS 内部使用此类型。

例如 defer 的返回类型为 Observable<ObservedValueOf<R>>，其中 R 是工厂函数返回的 ObservableInput。

TypeScript 示例

import { of, from, ObservedValueOf } from 'rxjs'; type A = ObservedValueOf<typeof of(1)>; // number type B = ObservedValueOf<Promise<string>>; // string type C = ObservedValueOf<number[]>; // number // defer 工厂返回 ObservableInput，ObservedValueOf 提取 T function defer<R extends ObservableInput<any>>(factory: () => R) : Observable<ObservedValueOf<R>> { ... }
