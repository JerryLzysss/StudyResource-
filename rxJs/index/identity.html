RxJS identity

核心概念：identity 是一个恒等函数，接收一个参数并原样返回，常用于 pipe 中作为默认或占位函数。

签名

identity<T>(x: T): T

参量

  • x（类型 T）：输入值

返回值

类型 T — 与输入相同的值。

描述

identity 是最简单的函数之一：传入什么就返回什么。在 RxJS 内部，它常作为默认的投影函数或参数转换函数的占位符。例如 fromEventPattern 的默认 resultSelector 就是只取第一个参数的 identity 行为。

例子

import { identity } from 'rxjs'; console.log(identity(42)); // 42 console.log(identity('hello')); // 'hello' console.log(identity({ a: 1 })); // { a: 1 }
