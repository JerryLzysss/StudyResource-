RxJS iif

核心概念：iif 在订阅时检查布尔条件，根据结果选择订阅 trueResult 或 falseResult 两个 Observable 源之一，充当条件分支的代理。

签名

iif<T, F>( condition: () => boolean, trueResult: ObservableInput<T>, falseResult: ObservableInput<F> ): Observable<T | F>

参量

  • condition：返回 boolean 的函数，决定订阅哪个 Observable
  • trueResult：条件为 true 时订阅的 Observable
  • falseResult：条件为 false 时订阅的 Observable

描述

iif 接受一个返回布尔值的 condition 函数，以及 trueResult 和 falseResult 两个源，返回一个 Observable。在订阅时刻，condition 函数被调用：若结果为 true，则订阅 trueResult；否则订阅 falseResult。

若需要在两个以上 Observable 之间选择，请查看 defer 创建方法。实际上 iif 内部基于 defer 实现，仅为方便和可读性而存在。

注意：作为参数传入的 Observable 表达式会被 eagerly 求值，但只会订阅被选中的那个分支。

例子

运行时切换订阅的 Observable

import { iif, of } from 'rxjs'; let subscribeToFirst; const firstOrSecond = iif( () => subscribeToFirst, of('first'), of('second') ); subscribeToFirst = true; firstOrSecond.subscribe(value => console.log(value)); // 输出： 'first' subscribeToFirst = false; firstOrSecond.subscribe(value => console.log(value)); // 输出： 'second'

控制对 Observable 的访问

import { iif, of, EMPTY } from 'rxjs'; let accessGranted; const observableIfYouHaveAccess = iif( () => accessGranted, of('It seems you have an access...'), EMPTY ); accessGranted = true; observableIfYouHaveAccess.subscribe({ next: value => console.log(value), complete: () => console.log('结束') }); // 输出： 'It seems you have an access...' / 'The end'
