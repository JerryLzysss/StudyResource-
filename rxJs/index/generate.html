RxJS generate

核心概念：generate 通过运行状态驱动的循环来生成 Observable 序列，类似 for 循环，在每次迭代时发出一个值。

签名

generate<T, S>( initialStateOrOptions: S | GenerateOptions<T, S>, condition?: ConditionFunc, iterate?: IterateFunc, resultSelectorOrObservable?: SchedulerLike | ResultFunc<S, T>, scheduler?: SchedulerLike ): Observable<T>

参量

  • initialState：初始状态
  • condition：终止条件（返回 false 时停止）
  • iterate：迭代步骤函数，更新状态
  • resultSelector（可选）：选择器函数，映射每次迭代的结果
  • scheduler（可选）：控制循环何时进行下一次迭代

描述

generate 允许您创建使用循环生成的值流，与传统的 for 循环非常相似：

  • 第一个参数是初始值（对应 for 循环的初始化表达式）
  • 第二个参数测试是否继续循环（对应 for 循环的条件表达式）
  • 第三个参数定义每步如何修改状态（对应 for 循环的递增表达式）

返回的 Observable 在每次循环迭代时发出一个值。首先运行条件函数，若返回 true，则发出当前存储的值，然后使用迭代函数更新该值；若条件返回 false，则 Observable 完成。

也可传入包含 initialState、condition、iterate、resultSelector 属性的选项对象。若省略 condition，则条件始终成立，Observable 永不完成。

默认情况下同步发出值；传入 async 调度器可确保每个值在事件循环的单独任务中推送。

例子

import { generate } from 'rxjs'; const source = generate( 0, x => x < 3, x => x + 1, x => x ); source.subscribe({ next: x => console.log('下一个: ' + x), complete: () => console.log('已完成') }); // => Next: 0 // => Next: 1 // => Next: 2 // => Completed

使用选项对象

import { generate } from 'rxjs'; const source = generate({ initialState: 0, condition: x => x < 5, iterate: x => x + 1, resultSelector: x => x * 2 });
