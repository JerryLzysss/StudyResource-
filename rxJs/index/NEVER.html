RxJS NEVER

核心概念：NEVER 是一个常量 Observable，不向观察者发出任何 next 值，也不触发 error 或 complete 通知。应使用 NEVER 常量，而非已弃用的 never() 函数。

声明

const NEVER: Observable<never>;

描述

NEVER 创建的 Observable 既不发出值，也不发出错误或完成通知。可用于测试，或与其他 Observable 组合。

由于从不发出 complete 通知，订阅 NEVER 不会自动清理，需要手动调用 unsubscribe()。

与 EMPTY 的区别：EMPTY 立即 complete；NEVER 永不 complete 也永不发出值。

例子

发出 7，然后不再发出任何值（连 complete 也没有）

import { NEVER } from 'rxjs'; import { startWith } from 'rxjs/operators'; function info() { console.log('不会被调用'); } const result = NEVER.pipe(startWith(7)); result.subscribe(x => console.log(x), info, info); // 输出: 7（之后 info 不会被调用）
