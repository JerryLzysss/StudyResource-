RxJS UnaryFunction

核心概念：UnaryFunction 是运算符接口，表示接受一个参数并返回一个值的函数类型，是 RxJS 管道类型系统的基础。

接口定义

interface UnaryFunction<T, R> { (source: T): R }

子接口

  • OperatorFunction<T, R>
  • MonoTypeOperatorFunction<T>（T → T 的特例）

方法

调用签名说明
(source: T): R接受类型 T 的输入，返回类型 R 的结果

描述

UnaryFunction 描述一种通用的一元函数类型。在 RxJS 中，OperatorFunction<T, R> 继承自 UnaryFunction，表示接受 Observable<T> 并返回 Observable<R> 的函数。pipe 方法正是利用这一类型系统来组合多个运算符。

例子

import { of } from 'rxjs'; import { map, filter } from 'rxjs/operators'; // map 是 OperatorFunction<number, string> // filter 是 MonoTypeOperatorFunction<number>（UnaryFunction 的子接口） of(1, 2, 3, 4).pipe( filter(x => x % 2 === 0), // UnaryFunction: Observable<number> → Observable<number> map(x => `偶数: ${x}`) // UnaryFunction: Observable<number> → Observable<string> ).subscribe(console.log); // 输出: 偶数: 2, 偶数: 4
