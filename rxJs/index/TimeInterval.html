RxJS TimeInterval

核心概念：TimeInterval 是一个接口，表示带有时间间隔信息的对象，包含原始值和距上次发射的毫秒数，由 timeInterval 运算符使用。

接口定义

interface TimeInterval<T> { value: T interval: number }

属性

属性类型描述
valueT源 Observable 发出的原始值
intervalnumber距上一次发射所经过的毫秒数

描述

TimeInterval 本身不是运算符，而是 timeInterval 运算符输出值的类型。每次源 Observable 发出新值时，timeInterval 会将其包装为 { value, interval } 对象，其中 interval 记录两次发射之间的时间差。

例子

import { interval } from 'rxjs'; import { timeInterval, take } from 'rxjs/operators'; interval(1000).pipe( timeInterval(), take(3) ).subscribe(ti => { console.log(`值: ${ti.value}, 间隔: ${ti.interval}ms`); }); // 输出（近似）: // 值: 0, 间隔: 1000ms // 值: 1, 间隔: 1000ms // 值: 2, 间隔: 1000ms
