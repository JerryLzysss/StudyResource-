RxJS partition（创建函数）

核心概念：partition 是 index 中的静态创建函数（非 operators 里已弃用的 partition 运算符），将源 Observable 分为两个：一个发出满足谓词的值，另一个发出不满足谓词的值。

签名

partition<T>( source: ObservableInput<T>, predicate: (value: T, index: number) => boolean, thisArg?: any ): [Observable<T>, Observable<T>]

参量

  • source：源 Observable（或任何可观察输入）
  • predicate：评估每个值的函数。返回 true 的值进入第一个 Observable，返回 false 的进入第二个。index 是从订阅以来第 i 次发射的编号（从 0 开始）
  • thisArg（可选）：谓词函数中 this 的值，默认 undefined

返回值

[Observable, Observable]：两个 Observable 的数组，第一个类似 filter 结果，第二个类似 filter 取反结果。

描述

类似于 filter，但一次返回两个 Observable。第一个发出谓词返回 true 的源值；第二个发出谓词返回 false 的源值。

注意：operators 中的 partition 已弃用，请使用此 index 静态创建函数：import { partition } from 'rxjs'。

例子

将数字划分为奇数和偶数

import { of, partition } from 'rxjs'; const observableValues = of(1, 2, 3, 4, 5, 6); const [evens$, odds$] = partition( observableValues, (value, index) => value % 2 === 0 ); odds$.subscribe(x => console.log('奇数', x)); evens$.subscribe(x => console.log('偶数', x)); // 输出: // odds 1 // odds 3 // odds 5 // evens 2 // evens 4 // evens 6
