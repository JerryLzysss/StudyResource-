RxJS AsyncSubject

核心概念：AsyncSubject 是 Subject 的变体，仅在完成时才发出值。完成后，它将向所有观察者发出其最新值。

类定义

class AsyncSubject<T> extends Subject { _subscribe(subscriber: Subscriber<any>): Subscription next(value: T): void error(error: any): void complete(): void }

说明

AsyncSubject 只会保存最后一个值，并且只在调用 complete() 时才将该值发送给所有订阅者。如果在完成前订阅，订阅者会等到 Subject 完成才收到值。

典型用途：HTTP 请求结果（只关心最终响应）、一次性异步操作的结果分发。

代码示例

import { AsyncSubject } from 'rxjs'; const subject = new AsyncSubject(); subject.subscribe(v => console.log('观察者 A: ' + v)); subject.next(1); subject.next(2); subject.next(3); subject.subscribe(v => console.log('观察者 B: ' + v)); subject.complete(); // 输出: // 观察者 A: 3 // 观察者 B: 3

方法

方法说明
_subscribe(subscriber)订阅 AsyncSubject，返回 Subscription
next(value: T)发送值（暂存最后一个，不立即通知订阅者）
error(error)发送错误通知
complete()完成并发出最后一个值
