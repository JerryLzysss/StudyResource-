RxJS NextObserver

核心概念：NextObserver 是 PartialObserver 的一种，必须提供 next 回调，用于处理 Observable 发出的值。

接口

interface NextObserver<T> { closed?: boolean next: (value: T) => void error?: (err: any) => void complete?: () => void }

属性

属性类型说明
closedboolean可选，表示订阅是否已关闭
next(value: T) => void必需，处理 next 通知
error(err: any) => void可选
complete() => void可选

描述

NextObserver 与 ErrorObserver、CompletionObserver 共同组成 PartialObserver<T> 联合类型。订阅 Observable 时，可以传入只包含 next 回调的 NextObserver；若未提供 error 或 complete 回调，对应通知会被忽略。

例子

import { of } from 'rxjs'; const nextObserver = { next: value => console.log('收到值:', value) }; of(1, 2, 3).subscribe(nextObserver); // 输出: 收到值: 1, 2, 3（complete 未提供，不打印完成信息）
