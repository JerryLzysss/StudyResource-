RxJS race（创建函数）

核心概念：race 是 index 中的创建函数，返回一个 Observable，它镜像第一个发出值的源 Observable。

签名

race(...sources: ObservableInput[]): Observable<T>

参量

  • sources：要竞争的 Observable 输入源（可传入多个）

描述

race 返回一个 Observable，订阅时会立即订阅所有源 Observable。一旦其中一个源 Observable 发出值，结果 Observable 就会取消订阅其他源。结果 Observable 将转发"获胜"源 Observable 的所有通知，包括 error 和 complete。

如果某个源 Observable 在发出第一个通知之前就抛出错误，race 也会抛出错误，无论其他源 Observable 是否有可能赢得竞争。

race 可用于从最快的 HTTP 或 WebSocket 网络连接中选择响应，也可用于根据用户输入切换 Observable 上下文。

例子

订阅第一个开始发射的 Observable

import { race, interval } from 'rxjs'; import { mapTo } from 'rxjs/operators'; const obs1 = interval(1000).pipe(mapTo('fast one')); const obs2 = interval(3000).pipe(mapTo('medium one')); const obs3 = interval(5000).pipe(mapTo('slow one')); race(obs3, obs1, obs2).subscribe(winner => console.log(winner)); // 输出: // 一系列 'fast one'
