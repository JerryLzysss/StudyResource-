RxJS ObservableInput

核心概念：ObservableInput 是 TypeScript 类型别名，表示可被 RxJS 转换为 Observable 的输入类型，包括 Subscribable、Promise、数组和可迭代对象。

类型定义

type ObservableInput<T> = | SubscribableOrPromise<T> | ArrayLike<T> | Iterable<T>;

描述

许多创建函数和操作符（如 from、combineLatest、concat、merge、forkJoin）接受 ObservableInput 作为参数。RxJS 内部会将这些输入统一转换为 Observable。

ObservableInput 包含：

  • Observable / Subscribable — 带 subscribe 方法的对象
  • Promise — 异步单值
  • ArrayLike / Iterable — 数组、Set、Map 等可迭代结构
  • InteropObservable — 实现 [Symbol.observable] 的互操作对象

例子

import { from, combineLatest, of } from 'rxjs'; // Promise 作为 ObservableInput from(Promise.resolve(42)).subscribe(console.log); // 数组作为 ObservableInput from([1, 2, 3]).subscribe(console.log); // 多个 ObservableInput 传给 combineLatest combineLatest(of('A'), [1, 2], Promise.resolve(true)) .subscribe(console.log);
