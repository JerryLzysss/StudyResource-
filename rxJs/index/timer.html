RxJS timer

核心概念：timer 创建一个 Observable，在 dueTime 之后开始发射，此后每隔 period 发射递增的数字序列。

签名

timer(dueTime: number | Date = 0, periodOrScheduler?: number | SchedulerLike, scheduler?: SchedulerLike): Observable<number>

参量

参数说明
dueTime可选，默认 0。指定为 Date 对象或整数，表示发出第一个值 0 之前要等待的毫秒数
periodOrScheduler可选。两次发射之间的时间间隔（后续编号）
scheduler可选。用于调度值发射的 SchedulerLike

描述

就像 interval，但是您可以指定何时应该开始发射。

timer 返回一个 Observable，它发出一个无限的升序整数序列，具有固定时间间隔 period。第一次发射发生在指定的时间 dueTime 之后。如果 period 未指定，则 Observable 仅发出一个值 0 后完成。

例子

发出升序数字，3 秒后开始，每秒一次

import { timer } from 'rxjs'; const numbers = timer(3000, 1000); numbers.subscribe(x => console.log(x));

五秒钟后发出一个数字

import { timer } from 'rxjs'; const numbers = timer(5000); numbers.subscribe(x => console.log(x));
