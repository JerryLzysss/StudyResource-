RxJS range

核心概念：range 创建一个 Observable，在指定范围内按顺序发出整数序列。

签名

range(start: number, count?: number, scheduler?: SchedulerLike): Observable<number>

参量

  • start：范围的起始数字
  • count（可选）：要发出的数字个数
  • scheduler（可选）：用于调度值发射的 SchedulerLike

描述

range 操作符按顺序发出一系列整数，您可以选择范围的起始值 start 及其长度 count。默认情况下不使用 SchedulerLike，同步地传递通知；也可以传入可选的 SchedulerLike 来调节传递时机。

例子

产生一个数字范围

import { range } from 'rxjs'; const numbers = range(1, 3); numbers.subscribe({ next: value => console.log(value), complete: () => console.log('完成！') }); // 输出: // 1 // 2 // 3 // 完成！
