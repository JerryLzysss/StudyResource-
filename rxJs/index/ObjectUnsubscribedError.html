RxJS ObjectUnsubscribedError

核心概念：ObjectUnsubscribedError 是在已取消订阅（unsubscribed）的对象上执行操作时抛出的错误，常见于对已 unsubscribe() 的 Subject 再次调用 next。

接口

interface ObjectUnsubscribedError extends Error { }

描述

当 Subject 或其他可订阅对象已经关闭（closed）后，若仍尝试向其推送值或订阅，RxJS 会抛出 ObjectUnsubscribedError。这是 RxJS 的安全机制，防止向已销毁的订阅者发送数据。

例子

import { Subject } from 'rxjs'; const subject = new Subject(); const sub = subject.subscribe(v => console.log(v)); sub.unsubscribe(); subject.next(1); // 抛出 ObjectUnsubscribedError
