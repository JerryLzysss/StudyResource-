RxJS merge（创建函数）

核心概念：merge 是 index 中的创建函数（非 operators 里已弃用的 merge 运算符），将多个 Observable 的值合并到一个输出 Observable 中同时发出。

签名

merge<T, R>(...observables: any[]): Observable<R>

参量

  • observables：要合并在一起的输入 Observable
  • concurrent（可选）：同时运行的最大 Observable 数量
  • scheduler（可选）：SchedulerLike 调度器

描述

通过将多个 Observable 的值混合到一个 Observable 中，将其平展在一起。

merge 订阅每个给定的输入 Observable（作为参数），并简单地将所有值从所有输入 Observable 转发（不进行任何转换）到输出 Observable。仅当所有输入 Observable 完成后，输出 Observable 才完成。输入 Observable 传递的任何错误将立即在输出 Observable 上发出。

例子

合并两个 Observable：1s 间隔和点击

import { merge, fromEvent, interval } from 'rxjs'; const clicks = fromEvent(document, 'click'); const timer = interval(1000); const clicksOrTimer = merge(clicks, timer); clicksOrTimer.subscribe(x => console.log(x)); // timer 每秒发出递增数字 // clicks 每次点击 document 时记录 MouseEvent // 合并后两者的事件按发生顺序交错输出

合并 3 个 Observable，但只有 2 个同时运行

import { merge, interval } from 'rxjs'; import { take } from 'rxjs/operators'; const timer1 = interval(1000).pipe(take(10)); const timer2 = interval(2000).pipe(take(6)); const timer3 = interval(500).pipe(take(10)); const concurrent = 2; const merged = merge(timer1, timer2, timer3, concurrent); merged.subscribe(x => console.log(x));
