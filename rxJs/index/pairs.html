RxJS pairs

核心概念：pairs 将对象、数组或 Map 等转换为 Observable，依次发出 [键, 值] 形式的二元组。

签名

pairs<O extends Record<string, unknown>>( obj: O, scheduler?: SchedulerLike ): Observable<[keyof O, O[keyof O]]>

参量

  • obj：对象、数组、Iterable 或 Map 等可枚举结构
  • scheduler（可选）：SchedulerLike 调度器

返回值

Observable<[string, T]>：发出键值对数组的 Observable。

描述

pairs 遍历对象的每个可枚举属性，将每个键值对作为 [key, value] 发出。对普通对象，键为字符串；对 Map，键可以是任意类型。

弃用说明：建议使用 from(Object.entries(obj)) 替代，v8 中将移除 pairs。

例子

将对象转换为键值对序列

import { pairs } from 'rxjs'; pairs({ a: 1, b: 2, c: 3 }).subscribe( x => console.log(x), err => console.log(err), () => console.log('完成') ); // 输出: // ['a', 1] // ['b', 2] // ['c', 3] // complete

也可以看看

  • from
  • pairwise（operators，连续两项配对）
  • Object.entries
