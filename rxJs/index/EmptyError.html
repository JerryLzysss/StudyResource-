RxJS EmptyError

核心概念：EmptyError 是在查询 Observable 或序列但没有元素时抛出的错误，常见于 first、last、single 等运算符。

接口

interface EmptyError extends Error { }

描述

当 Observable 完成时未发出任何值，而运算符要求至少有一个元素时，会抛出 EmptyError。例如对空的 Observable 使用 first() 或 single()。

例子

import { EMPTY } from 'rxjs'; import { first } from 'rxjs/operators'; EMPTY.pipe(first()).subscribe({ next: v => console.log(v), error: err => console.error(err.name) // EmptyError });
