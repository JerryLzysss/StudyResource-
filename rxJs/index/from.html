RxJS from

核心概念：from 从数组、类数组对象、Promise、可迭代对象或类 Observable 对象创建 Observable，几乎可将任何数据源转换为 Observable。

签名

from<T>(input: any, scheduler?: SchedulerLike): Observable<T>

参量

参数说明
input类型：any。数组、Promise、Iterable 等
scheduler可选。默认 undefined。类型：SchedulerLike

返回值

Observable<T>

描述

from 将各种其他对象和数据类型转换为 Observable。它还将 Promise、类数组对象或可迭代对象转换为 Observable，该 Observable 发出 Promise、数组或可迭代的各项。字符串被视为字符数组。包含 ES2015 Symbol.observable 的类 Observable 对象也可通过此运算符转换。

例子

将数组转换为 Observable

import { from } from 'rxjs'; const array = [10, 20, 30]; const result = from(array); result.subscribe(x => console.log(x)); // 输出： // 10 // 20 // 30

将无限迭代（从生成器）转换为 Observable

import { from } from 'rxjs'; import { take } from 'rxjs/operators'; function* generateDoubles(seed) { let i = seed; while (true) { yield i; i = 2 * i; } } const iterator = generateDoubles(3); const result = from(iterator).pipe(take(10)); result.subscribe(x => console.log(x)); // 输出： // 3, 6, 12, 24, 48, 96, 192, 384, 768, 1536

使用异步调度程序

import { from, asyncScheduler } from 'rxjs'; console.log('开始'); const array = [10, 20, 30]; const result = from(array, asyncScheduler); result.subscribe(x => console.log(x)); console.log('结束'); // 输出： // start // end // 10 // 20 // 30
