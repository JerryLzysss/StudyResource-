RxJS fromEvent

核心概念：fromEvent 从给定事件目标（DOM、Node.js EventEmitter 等）创建 Observable，在订阅时注册事件监听，取消订阅时移除。

签名

fromEvent<T>( target: any, eventName: string, options?: EventListenerOptions | ((...args: any[]) => T), resultSelector?: (...args: any[]) => T ): Observable<T>

描述

创建一个 Observable，它从给定事件目标发出特定类型的事件。

fromEvent 接受事件目标（具有注册事件处理函数方法的对象）和事件类型字符串。若事件目标不在支持列表中，应使用 fromEventPattern。

每次订阅时，事件处理函数会注册到目标上；事件触发时，处理函数的第一个参数会被 emit；取消订阅时，处理函数会被移除。

支持的事件目标

  • DOM EventTarget：具有 addEventListener / removeEventListener
  • Node.js EventEmitter：具有 addListener / removeListener
  • JQuery 风格：具有 on / off 方法
  • NodeList / HTMLCollection：为每个节点安装/移除监听

例子

发出 document 上发生的点击

import { fromEvent } from 'rxjs'; const clicks = fromEvent(document, 'click'); clicks.subscribe(x => console.log(x)); // 每次在 document 上点击，控制台输出 MouseEvent 对象

使用 addEventListener 的 capture 选项

import { fromEvent } from 'rxjs'; const div = document.createElement('div'); document.body.appendChild(div); const clicksInDocument = fromEvent(document, 'click', { capture: true }); const clicksInDiv = fromEvent(div, 'click'); clicksInDocument.subscribe(() => console.log('文档')); clicksInDiv.subscribe(() => console.log('div 元素')); // 使用 capture 时，document 在捕获阶段先收到事件
