RxJS InteropObservable

核心概念：InteropObservable 描述遵循 Observable 互操作规范的对象，通过 Symbol.observable 提供可订阅接口，用于与第三方库集成。

类型定义

type InteropObservable<T> = { [Symbol.observable]: () => Subscribable<T>; };

描述

InteropObservable 是 RxJS 中用于描述「类 Observable」对象的 TypeScript 类型。任何实现了 [Symbol.observable] 方法并返回 Subscribable 的对象，都可以被 RxJS 的 from 等函数识别并转换。

这是 ECMAScript Observable 互操作提案的标准方式，取代了已弃用的 ObservableLike 类型。

RxJS 内部使用此类型来判断某个值是否可以作为 Observable 输入（ObservableInput）被订阅。

例子

import { from } from 'rxjs'; // 自定义 InteropObservable 对象 const customInterop = { [Symbol.observable]() { return { subscribe(observer) { observer.next('来自 InteropObservable'); observer.complete(); return { unsubscribe() {} }; } }; } }; from(customInterop).subscribe(console.log); // 输出: '来自 InteropObservable'
