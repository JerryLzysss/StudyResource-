RxJS Observer

核心概念：Observer 是 Observable 传递的值的消费者，包含 next、error、complete 三个必需回调，用于处理 Observable 发出的各类通知。

接口

interface Observer<T> { closed?: boolean next: (value: T) => void error: (err: any) => void complete: () => void }

类的实现

  • Subscriber

属性

属性类型
closedboolean（可选）
next(value: T) => void
error(err: any) => void
complete() => void

描述

Observer 是 RxJS 观察者模式的核心类型。订阅 Observable 时传入 Observer，Observable 会在执行过程中调用对应的回调。完整的 Observer 必须同时提供 next、error、complete 三个方法。

若只需部分回调，可使用 PartialObserver（如 NextObserver、ErrorObserver、CompletionObserver）。

例子

import { of } from 'rxjs'; const observer = { next: x => console.log('next:', x), error: err => console.error('error:', err), complete: () => console.log('完成') }; of(1, 2, 3).subscribe(observer);
