RxJS FactoryOrValue

核心概念：FactoryOrValue 是一个类型别名，表示「值本身」或「返回该值的工厂函数」，常用于需要延迟求值或灵活传参的 API。

类型别名

type FactoryOrValue<T> = T | (() => T);

描述

当 API 接受 FactoryOrValue<T> 时，你可以直接传入类型为 T 的值，也可以传入无参函数 () => T。传入工厂函数时，通常会在需要时才调用以获取实际值，从而实现惰性创建或每次获取新实例。

典型场景包括 iif 的条件分支、某些运算符的可选配置，以及需要在订阅时才创建 Subject 的多播场景。

例子

// 直接传值 const value: FactoryOrValue<number> = 42; // 传工厂函数（延迟求值） const factory: FactoryOrValue<number> = () => Math.random(); function resolve<T>(input: FactoryOrValue<T>): T { return typeof input === 'function' ? input() : input; } console.log(resolve(value)); // 42 console.log(resolve(factory)); // 随机数
