RxJS scheduled

核心概念：scheduled 将常见的 ObservableInput 类型转换为 Observable，并在指定的调度程序上安排订阅和发射。

签名

scheduled<T>(input: ObservableInput<T>, scheduler: SchedulerLike): Observable<T>

参量

  • input：要调度的 Observable、数组、Promise、Iterable 等 ObservableInput
  • scheduler：用于安排订阅和发射的 SchedulerLike 调度程序

描述

scheduled 是 RxJS 内部用于将各种输入源统一转换为 Observable 并指定调度时机的工具函数。它接受任意 ObservableInput（如数组、Promise、Iterable）和 SchedulerLike，返回在该调度器上执行订阅与发射的 Observable。

from 和 of 等创建函数在传入 scheduler 参数时，内部会使用 scheduled 来实现。

例子

import { scheduled, of, asyncScheduler } from 'rxjs'; // 将 of 的值在 async 调度器上异步发射 const result = scheduled(of(1, 2, 3), asyncScheduler); result.subscribe(v => console.log(v)); // 等价于: // of(1, 2, 3, asyncScheduler).subscribe(...)
