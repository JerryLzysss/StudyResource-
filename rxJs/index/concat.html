RxJS concat

核心概念：concat 按顺序连接多个 Observable，先订阅第一个并发出其全部值，待其完成后才订阅下一个，依次类推。

签名

concat(...observables: ObservableInput[]): Observable<unknown>

描述

concat 将多个 Observable 按顺序连接在一起，通过依次订阅它们并将结果合并到输出 Observable 中。可以传递 Observable 数组，也可以直接作为参数传入。传递空数组将导致 Observable 立即完成。

concat 将订阅第一个输入 Observable 并发出其所有值，不会以任何方式改变它们。当该 Observable 完成时，它将订阅下一个传入的 Observable 并再次发出其值。此过程重复，直到运算符用完所有 Observable。当最后一个输入 Observable 完成时，concat 也将完成。

在任何给定时刻，只有传递给 concat 的一个 Observable 会发出值。如果希望并发发出值，请使用 merge。

注意：如果某个输入 Observable 永不完成，concat 也永远不会完成，其后的 Observable 将永远不会被订阅。如果链中任何 Observable 出错，concat 将立即出错。

例子

按顺序连接多个 Observable

import { concat, of } from 'rxjs'; concat( of(1, 2, 3), of(4, 5, 6), of(7, 8, 9) ).subscribe(console.log); // 输出: 1, 2, 3, 4, 5, 6, 7, 8, 9

与 throwError 组合

import { concat, of, throwError } from 'rxjs'; const result = concat(of(7), throwError(() => new Error('oops!'))); result.subscribe( x => console.log(x), e => console.error(e) ); // 输出: // 7 // Error: oops!
