RxJS ObservedValuesFromArray

核心概念：ObservedValuesFromArray 是 TypeScript 条件类型，从 ObservableInput[] 数组中提取每个输入对应的值类型元组。

类型定义

type ObservedValuesFromArray<X> = X extends Array<ObservableInput<infer T>> ? T : never;

描述

ObservedValuesFromArray 用于 combineLatest、forkJoin、zip 等接受多个 ObservableInput 的函数，在 TypeScript 中推断合并结果的元组类型。

例如传入 [of(1), of('a')]，ObservedValuesFromArray 帮助推断结果为 [number, string]。

TypeScript 示例

import { combineLatest, of } from 'rxjs'; const sources = [of(1), of('hello')] as const; combineLatest(sources).subscribe(([num, str]) => { // num: number, str: string console.log(num, str); }); // combineLatest 签名中使用 ObservedValuesFromArray // 推断 (...values: ObservedValuesFromArray<O>[]) => R
