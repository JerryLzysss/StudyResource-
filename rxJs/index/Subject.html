RxJS Subject

核心概念：Subject 是一种特殊的 Observable，允许将值多播到许多 Observer。它既是 Observable 又是 Observer，类似 EventEmitter。

类定义

class Subject<T> extends Observable implements SubscriptionLike { static create: Function constructor() observers: Observer<T>[] closed: boolean isStopped: boolean hasError: boolean thrownError: any lift<R>(operator: Operator<T, R>): Observable<R> next(value?: T): void error(err: any): void complete(): void unsubscribe(): void asObservable(): Observable<T> }

描述

每个 Subject 都是 Observable 和 Observer。您可以订阅 Subject，也可以调用 next 来推送值，以及 error 和 complete。

普通的 Observable 是单播的（每个订阅的 Observer 拥有独立执行），而 Subject 是多播的，它将值广播给所有已注册的观察者。

Subject 的几个特例：BehaviorSubject、ReplaySubject 和 AsyncSubject。

方法

方法说明
next(value)向所有订阅者多播新值
error(err)向所有订阅者发送错误通知
complete()向所有订阅者发送完成通知
asObservable()将此 Subject 作为源创建新 Observable，隐藏 Subject 端逻辑
unsubscribe()取消 Subject 本身

例子

多播到多个观察者

import { Subject } from 'rxjs'; const subject = new Subject<number>(); subject.subscribe({ next: v => console.log(`observerA: ${v}`) }); subject.subscribe({ next: v => console.log(`observerB: ${v}`) }); subject.next(1); subject.next(2); // 输出： // observerA: 1 // observerB: 1 // observerA: 2 // observerB: 2

将单播 Observable 转为多播

import { Subject, from } from 'rxjs'; const subject = new Subject<number>(); subject.subscribe({ next: v => console.log(`observerA: ${v}`) }); subject.subscribe({ next: v => console.log(`observerB: ${v}`) }); const observable = from([1, 2, 3]); observable.subscribe(subject); // 输出： // observerA: 1 observerB: 1 // observerA: 2 observerB: 2 // observerA: 3 observerB: 3
