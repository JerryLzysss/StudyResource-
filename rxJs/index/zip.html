RxJS zip

核心概念：zip 是一个创建函数，将多个 Observable 按索引配对组合，等待每个源各发出一个值后将它们作为数组（或通过投影函数处理后）发出。

签名

zip(...sources: ObservableInput[]): Observable zip(...sources: ObservableInput[], resultSelector: (...values) => R): Observable<R>

参量

参数说明
...sources要组合的一个或多个 Observable 输入源
resultSelector可选。将各源的值合并为单个结果的投影函数

描述

zip 订阅所有传入的 Observable，等待每个源都发出一个值后，将这些值按索引配对发出。只要所有源 Observable 都还有值可发，zip 就会继续。任一源完成时，zip 也会完成。

注意：管道中的 zip 运算符已弃用，请改用 zipWith。此处介绍的是 index 目录下的创建函数 zip(...sources)。

例子

将两个 Observable 按索引配对

import { zip, of } from 'rxjs'; zip(of(1, 2, 3), of('a', 'b', 'c')).subscribe(console.log); // 输出： // [1, 'a'] // [2, 'b'] // [3, 'c']

使用投影函数

import { zip, of } from 'rxjs'; zip( of(1, 2), of('a', 'b'), (n, s) => n + s ).subscribe(console.log); // 输出： // '1a' // '2b'
