RxJS Timestamp

核心概念：Timestamp 是一个接口，表示带有时间戳信息的对象，包含原始值和发射时刻的时间戳，由 timestamp 运算符使用。

接口定义

interface Timestamp<T> { value: T timestamp: number }

属性

属性类型描述
valueT源 Observable 发出的原始值
timestampnumber值发射时的时间戳（毫秒，由调度器提供）

描述

Timestamp 本身不是运算符，而是 timestamp 运算符输出值的类型。每次源 Observable 发出新值时，timestamp 会将其包装为 { value, timestamp } 对象。

例子

import { of } from 'rxjs'; import { timestamp } from 'rxjs/operators'; of('A', 'B', 'C').pipe( timestamp() ).subscribe(ts => { console.log(`值: ${ts.value}, 时间戳: ${ts.timestamp}`); });
