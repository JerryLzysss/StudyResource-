RxJS single

核心概念：single 断言源 Observable 只发出一个值，否则抛出 EmptyError（0 个值）或 SequenceError（多个值）。

签名

single<T>(predicate?: (value: T, index: number, source: Observable<T>) => boolean): MonoTypeOperatorFunction<T>

参量

  • predicate（可选）：断言函数，源必须恰好有一个值满足条件

返回值

返回 MonoTypeOperatorFunction，源 complete 时发出唯一值或 error。

描述

single 用于确保 Observable 只产生一个期望的值，常用于查询结果必须唯一的场景。

例子

import { of } from 'rxjs'; import { single } from 'rxjs/operators'; of(42).pipe(single()).subscribe(console.log); // 输出: 42 of(1, 2).pipe(single()).subscribe( v => console.log(v), err => console.log(err.name) // SequenceError );
