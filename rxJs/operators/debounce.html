RxJS debounce

核心概念：debounce 忽略源 Observable 在指定时间窗口内连续发出的值，仅在静默期结束后才发出最后一个值。

签名

debounce<T>( durationSelector: (value: T) => ObservableInput<any> ): MonoTypeOperatorFunction<T>

描述

debounce 通过 durationSelector 函数为每个源值确定一个"静默期" Observable。当源发出新值时，会取消之前的计时并重新开始。只有当 durationSelector 返回的 Observable 发出值时，才会发出源的最新值。

与 debounceTime 不同，debounce 的延迟时间可以动态变化。实际开发中，基于固定时间的 debounceTime 更常用。

例子

import { of, timer } from 'rxjs'; import { debounce } from 'rxjs/operators'; of('等待', '一', '秒', '最后会显示').pipe( debounce(() => timer(1000)) ).subscribe(x => console.log(x)); // 输出: 最后会显示（1秒静默后发出最后一个值）
