RxJS startWith

核心概念：startWith 在源 Observable 发出值之前，先发出指定的初始值。

签名

startWith<T, D>(...values: D[]): OperatorFunction<T, T | D>

参量

  • values：要在源之前发出的初始值，可传多个

返回值

返回 OperatorFunction，先发出初始值再发出源的值。

描述

常用于为异步流提供初始状态，如 UI 组件的默认值、combineLatest 需要所有源都有初始值等。

例子

import { of } from 'rxjs'; import { startWith } from 'rxjs/operators'; of(2, 3, 4).pipe( startWith(1) ).subscribe(console.log); // 输出: 1, 2, 3, 4
