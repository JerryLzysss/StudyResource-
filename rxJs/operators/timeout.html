RxJS timeout

核心概念：timeout 在指定时间内未收到源的任何通知（next/error/complete）时，抛出 TimeoutError。

签名

timeout<T>(each?: number, scheduler?: SchedulerLike): MonoTypeOperatorFunction<T> timeout<T>(config: TimeoutConfig<T>): MonoTypeOperatorFunction<T>

参量

  • each：两次通知之间的最大间隔（毫秒）
  • config：{ each, with, scheduler } 高级配置

返回值

返回 MonoTypeOperatorFunction，超时时 error。

描述

常用于 HTTP 请求、WebSocket 心跳检测等需要超时控制的场景。

例子

import { timer } from 'rxjs'; import { timeout } from 'rxjs/operators'; timer(3000).pipe( timeout(1000) ).subscribe( v => console.log(v), err => console.log(err.name) // TimeoutError );
