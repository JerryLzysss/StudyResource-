RxJS endWith

核心概念：endWith 在源 Observable 完成后，追加发出指定的值，然后完成。与 startWith 相反。

签名

endWith<T, A extends unknown[]>( ...values: A ): OperatorFunction<T, T | A[number]>

描述

endWith 先转发源 Observable 的所有值。当源完成时，按顺序发出 endWith 参数中提供的值，然后完成。

常用于在流结束时追加收尾消息或默认值，例如 API 请求完成后追加一个"加载完成"状态。

例子

import { of } from 'rxjs'; import { endWith } from 'rxjs/operators'; of(1, 2).pipe( endWith(3, 4) ).subscribe(x => console.log(x)); // 输出: 1, 2, 3, 4
