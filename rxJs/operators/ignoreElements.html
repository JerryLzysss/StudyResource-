RxJS ignoreElements

核心概念：ignoreElements 忽略源 Observable 发出的所有 next 值，仅转发 error 和 complete 通知。

函数签名

ignoreElements(): OperatorFunction

说明

当你只关心 Observable 何时结束或出错，而不关心中间值时使用。

输出 Observable 不会发出任何 next 值，只会在源 complete 或 error 时发出相应通知。

代码示例

import { of } from 'rxjs'; import { ignoreElements, tap } from 'rxjs/operators'; of(1, 2, 3).pipe( tap(v => console.log('tap:', v)), ignoreElements() ).subscribe({ complete: () => console.log('仅收到 complete') }); // tap: 1, 2, 3（tap 仍执行） // 仅收到 complete
