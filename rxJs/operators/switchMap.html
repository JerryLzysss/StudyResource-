RxJS switchMap

核心概念：switchMap 将每个源值映射为内部 Observable，切换到最新的内部 Observable 并取消之前的，常用于搜索、自动完成等场景。

签名

switchMap<T, R, O extends ObservableInput<R>>( project: (value: T, index: number) => O ): OperatorFunction<T, ObservedValueOf<O>>

参量

  • project：将源值映射为内部 Observable 的函数

返回值

返回 OperatorFunction，展平内部 Observable 并切换到最新。

描述

与 mergeMap（并行）和 concatMap（串行）不同，switchMap 在新值到来时取消未完成的内部 Observable。典型用途：用户输入搜索、路由切换取消旧请求。

例子

import { of, interval } from 'rxjs'; import { switchMap, map, take } from 'rxjs/operators'; of('query1', 'query2', 'query3').pipe( switchMap(q => interval(300).pipe( map(i => `${q}-${i}`), take(3) )) ).subscribe(console.log); // 只输出 query3 的结果
