RxJS pluck

核心概念：pluck 从每个发出的对象中提取指定属性，等价于 map(x => x.property)，支持嵌套属性路径。

签名

pluck<T, K extends keyof T>(property: K): OperatorFunction<T, T[K]> pluck<T, K1, K2>(k1: K1, k2: K2): OperatorFunction<T, T[K1][K2]> // ... 最多支持 6 层嵌套

参量

  • property：要提取的属性名，或多个属性名组成的路径

返回值

返回 OperatorFunction，只发出指定属性的值。

描述

pluck 是 map 的便捷写法，专门用于从对象流中提取字段。若属性不存在，发出 undefined。

例子

import { from } from 'rxjs'; import { pluck } from 'rxjs/operators'; const users = [ { name: 'Alice', age: 25 }, { name: 'Bob', age: 30 } ]; from(users).pipe( pluck('name') ).subscribe(console.log); // 输出: 'Alice', 'Bob' // 嵌套属性 from([{ user: { name: 'Carol' } }]).pipe( pluck('user', 'name') ).subscribe(console.log); // 输出: 'Carol'
