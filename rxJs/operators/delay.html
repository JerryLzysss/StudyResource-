RxJS delay

核心概念：delay 延迟源 Observable 发出值的时间，每个 next 通知都会按指定的毫秒数或日期延迟后再发出。

签名

delay<T>( due: number | Date, scheduler?: SchedulerLike ): MonoTypeOperatorFunction<T>

描述

delay 运算符将源 Observable 发出的每个值延迟指定的时间后再转发给观察者。due 可以是毫秒数，也可以是 Date 对象（延迟到该时间点发出）。

注意：delay 不会延迟 error 和 complete 通知的时间（它们会立即转发），只延迟 next 通知。

例子

import { of, merge } from 'rxjs'; import { delay, mapTo } from 'rxjs/operators'; const source = of(null); merge( source.pipe(mapTo('你好')), source.pipe(mapTo('World'), delay(1000)), source.pipe(mapTo('Goodbye'), delay(2000)) ).subscribe(val => console.log(val)); // 输出: // 你好（立即） // World（1秒后） // Goodbye（2秒后）
