RxJS pairwise

核心概念：pairwise 将当前值与上一个值配对，以 [previous, current] 数组形式发出。第一个值会被跳过。

签名

pairwise<T>(): OperatorFunction<T, [T, T]>

返回值

返回 OperatorFunction，发出连续两个值的元组 [previous, current]。

描述

pairwise 在收到第二个值后才开始发出。常用于计算相邻值的差值、检测变化方向等场景。

例子

import { of } from 'rxjs'; import { pairwise } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( pairwise() ).subscribe(console.log); // 输出: // [1, 2] // [2, 3] // [3, 4] // [4, 5]
