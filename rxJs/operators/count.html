RxJS count

核心概念：count 统计源 Observable 发出的值的数量，在源完成时发出总计数。

签名

count<T>(predicate?: (value: T, index: number) => boolean): OperatorFunction<T, number>

描述

count 运算符订阅源 Observable，统计其发出的 next 通知数量。当源 Observable 完成时，发出一个包含总数的 single 值，然后完成。

可选的 predicate 函数用于过滤：只有满足条件的值才会被计入总数。

例子

import { of } from 'rxjs'; import { count } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( count() ).subscribe(x => console.log(x)); // 输出: 5 of(1, 2, 3, 4, 5).pipe( count(val => val % 2 === 0) ).subscribe(x => console.log(x)); // 输出: 2（偶数个数）
