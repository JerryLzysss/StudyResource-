RxJS timestamp

核心概念：timestamp 将每个源值包装为 Timestamp 对象，包含 value 和 timestamp（自 epoch 起的毫秒数）。

签名

timestamp<T>(scheduler?: SchedulerLike): OperatorFunction<T, Timestamp<T>>

参量

  • scheduler（可选）：Scheduler，默认 asyncScheduler

返回值

返回 OperatorFunction，发出 { value, timestamp } 对象。

描述

常用于日志记录、事件追踪、审计等需要知道值发出时间的场景。

例子

import { of } from 'rxjs'; import { timestamp } from 'rxjs/operators'; of('event1', 'event2').pipe( timestamp() ).subscribe(console.log); // { value: 'event1', timestamp: 1234567890123 } // { value: 'event2', timestamp: 1234567890124 }
