RxJS debounceTime

核心概念：debounceTime 忽略源 Observable 在指定毫秒数内连续发出的值，仅在静默期结束后才发出最后一个值。常用于搜索框输入防抖。

签名

debounceTime<T>( dueTime: number, scheduler?: SchedulerLike ): MonoTypeOperatorFunction<T>

描述

debounceTime 是 debounce 的时间版本，使用固定的 dueTime（毫秒）作为静默期。每当源发出新值，计时器会重置。只有当 dueTime 毫秒内没有新值发出时，才会发出最后一个值。

典型应用场景：用户输入搜索关键词时，等待用户停止输入 300ms 后再发起请求，避免频繁调用 API。

例子

import { fromEvent } from 'rxjs'; import { debounceTime, map } from 'rxjs/operators'; const input = document.getElementById('search'); fromEvent(input, 'input').pipe( map(e => e.target.value), debounceTime(500) ).subscribe(val => console.log('防抖输入:', val));
