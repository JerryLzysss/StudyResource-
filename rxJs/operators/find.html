RxJS find

核心概念：find 发出源 Observable 中第一个满足谓词条件的值，然后完成；若无匹配则发出 undefined 后完成。

函数签名

find(predicate: (value: T, index: number) => boolean): Observable

说明

类似 Array.prototype.find。找到第一个匹配值后立即 complete，不再接收后续值。

若整个序列结束仍未找到，则发出 undefined 并完成。

参数

参数说明
predicate查找条件，返回 true 的第一个值会被发出

代码示例

import { of } from 'rxjs'; import { find } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe(find(x => x > 3)) .subscribe(console.log); // 输出: 4
