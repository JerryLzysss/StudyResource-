RxJS filter

核心概念：filter 类似 Array.prototype.filter，仅发出满足谓词函数条件的值，不满足的值会被跳过。

函数签名

filter(predicate: (value: T, index: number) => boolean): MonoTypeOperatorFunction

说明

对源 Observable 发出的每个值执行 predicate 函数。返回 true 则向下游发出该值，返回 false 则忽略。

index 参数表示自订阅以来第几次发射（从 0 开始）。常用于数据筛选、事件过滤等场景。

参数

参数说明
predicate判断函数，返回 true 则发出该值
thisArg（可选）predicate 中 this 的指向（已弃用，建议用闭包）

代码示例

import { of } from 'rxjs'; import { filter } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe(filter(x => x % 2 === 0)) .subscribe(console.log); // 输出: 2, 4
