RxJS tap

核心概念：tap 对 Observable 的生命周期执行副作用（next/error/complete/subscribe/unsubscribe），不改变数据流。

签名

tap<T>(observerOrNext?: PartialObserver<T> | ((value: T) => void)): MonoTypeOperatorFunction<T>

参量

  • observerOrNext：回调函数或 PartialObserver 对象

返回值

返回 MonoTypeOperatorFunction，透传所有值，仅执行副作用。

描述

tap 常用于调试日志、记录指标、触发 UI 更新等，不影响下游接收的数据。

例子

import { of } from 'rxjs'; import { tap, map } from 'rxjs/operators'; of(1, 2, 3).pipe( tap(v => console.log('tap:', v)), map(v => v * 10) ).subscribe(console.log); // tap: 1, 10, tap: 2, 20, tap: 3, 30
