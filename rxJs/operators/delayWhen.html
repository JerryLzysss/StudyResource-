RxJS delayWhen

核心概念：delayWhen 根据 selector 函数返回的 Observable 来决定每个值的延迟时间，实现动态延迟。

签名

delayWhen<T>( delayDurationSelector: (value: T, index: number) => ObservableInput<any>, subscriptionDelay?: ObservableInput<any> ): MonoTypeOperatorFunction<T>

描述

delayWhen 为源 Observable 的每个值调用 delayDurationSelector，返回一个"延迟通知" Observable。只有当该 Observable 发出值时，源的值才会被转发。不同的值可以有不同的延迟策略。

可选的 subscriptionDelay 参数用于延迟整个订阅的开始时间。

例子

import { interval, timer } from 'rxjs'; import { delayWhen, take } from 'rxjs/operators'; interval(1000).pipe( take(3), delayWhen(() => timer(2000)) ).subscribe(x => console.log(x)); // 每个值延迟 2 秒后发出 // 输出: 0, 1, 2（每个间隔约 3 秒）
