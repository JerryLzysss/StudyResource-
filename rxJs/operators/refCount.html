RxJS refCount

核心概念：refCount 使 ConnectableObservable 在第一个订阅者订阅时自动 connect，最后一个订阅者取消订阅时自动 disconnect。

签名

refCount<T>(): MonoTypeOperatorFunction<T>

返回值

返回普通 Observable，自动管理 connect/disconnect 生命周期。

描述

refCount 通常跟在 publish、publishReplay 等之后使用。与 share() 等价于 publish().refCount()。

例子

import { interval } from 'rxjs'; import { publish, refCount, tap } from 'rxjs/operators'; const shared = interval(1000).pipe( tap(v => console.log('源:', v)), publish(), refCount() ); const sub1 = shared.subscribe(v => console.log('A:', v)); // 第一个订阅者触发 connect setTimeout(() => sub1.unsubscribe(), 3000); // 最后一个订阅者取消后 disconnect
