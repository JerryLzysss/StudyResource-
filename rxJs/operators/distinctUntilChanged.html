RxJS distinctUntilChanged

核心概念：distinctUntilChanged 只有当前值与上一个值不同时才发出，用于过滤连续重复的值。

签名

distinctUntilChanged<T, K>( compare?: (previous: K, current: K) => boolean, keySelector?: (value: T) => K ): MonoTypeOperatorFunction<T>

描述

distinctUntilChanged 比较当前值与上一个发出的值。如果相同则忽略，不同则发出。默认使用 === 比较。

与 distinct 不同，distinctUntilChanged 只比较相邻的值。例如序列 [1,1,2,2,1] 会输出 [1,2,1]。可选的 compare 和 keySelector 用于自定义比较逻辑。

例子

import { of } from 'rxjs'; import { distinctUntilChanged } from 'rxjs/operators'; of(1, 1, 2, 2, 3, 1, 2, 3).pipe( distinctUntilChanged() ).subscribe(x => console.log(x)); // 输出: 1, 2, 3, 1, 2, 3
