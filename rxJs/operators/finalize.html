RxJS finalize

核心概念：finalize 在源 Observable 终止时（complete、error 或取消订阅）调用指定的回调函数，常用于清理资源。

函数签名

finalize(callback: () => void): MonoTypeOperatorFunction

说明

返回一个镜像源 Observable 的新 Observable，但在源终止时会调用 callback。

无论正常完成、出错还是订阅者主动 unsubscribe，都会执行回调。类似 try/finally 中的 finally 块。

参数

参数说明
callback源终止时执行的函数

代码示例

import { interval, take, finalize } from 'rxjs'; interval(1000).pipe( take(3), finalize(() => console.log('序列结束')) ).subscribe(v => console.log(v)); // 0, 1, 2, '序列结束'
