RxJS CombineAll

核心概念：CombineAll 用于处理高阶 Observable。它等待源 Observable 完成，收集所有内部 Observable，然后像 combineLatest 一样合并它们的最新值。

签名

combineAll(): OperatorFunction<ObservableInput<T>, T[]>

描述

当源 Observable 发出内部 Observable 时，CombineAll 会缓存这些内部 Observable，但不会立即订阅它们。只有当源 Observable 完成（complete）后，才会对所有收集到的内部 Observable 应用 combineLatest 行为。

每当任意一个内部 Observable 发出新值时，CombineAll 会发出一个数组，包含每个内部 Observable 的最新值。典型场景：用户连续点击多次，每次点击产生一个 interval 流，等所有点击结束后，再组合这些 interval 的最新值。

与 combineLatest 的区别

  • combineLatest：静态创建函数，直接组合多个已知 Observable
  • combineAll：实例运算符，处理 Observable 的 Observable，等源完成后再组合

例子

import { fromEvent, interval } from 'rxjs'; import { map, combineAll, take } from 'rxjs/operators'; const clicks = fromEvent(document, 'click'); const result = clicks.pipe( take(2), map(() => interval(1000)), combineAll() ); result.subscribe(x => console.log(x)); // 点击两次后，源完成，开始组合两个 interval // 输出: [0, 0], [1, 0], [1, 1], [2, 1], ...
