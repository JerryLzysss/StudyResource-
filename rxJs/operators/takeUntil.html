RxJS takeUntil

核心概念：takeUntil 持续发出源的值，直到 notifier Observable 发出值，然后 complete 并取消订阅。

签名

takeUntil<T>(notifier: Observable<any>): MonoTypeOperatorFunction<T>

参量

  • notifier：发出值时终止源的 Observable

返回值

返回 MonoTypeOperatorFunction，notifier 发出值后 complete。

描述

常用于组件销毁时取消订阅：source.pipe(takeUntil(destroy$))。

例子

import { interval, timer } from 'rxjs'; import { takeUntil } from 'rxjs/operators'; interval(300).pipe( takeUntil(timer(1500)) ).subscribe(console.log); // 约 1.5 秒后停止
