RxJS defaultIfEmpty

核心概念：defaultIfEmpty 在源 Observable 为空（没有发出任何值就完成）时，发出指定的默认值。

签名

defaultIfEmpty<T, D>( defaultValue: D ): OperatorFunction<T, T | D>

描述

如果源 Observable 在完成前至少发出过一个值，defaultIfEmpty 会原样转发所有值。如果源 Observable 直接完成而没有发出任何值，则发出 defaultValue 作为唯一值，然后完成。

常用于确保 Observable 链始终有一个可订阅的结果，避免空流导致下游逻辑无法执行。

例子

import { EMPTY, of } from 'rxjs'; import { defaultIfEmpty } from 'rxjs/operators'; of().pipe( defaultIfEmpty('Observable.of() 为空!') ).subscribe(x => console.log(x)); // 输出: Observable.of() 为空! EMPTY.pipe( defaultIfEmpty('EMPTY 的默认值') ).subscribe(x => console.log(x)); // 输出: EMPTY 的默认值
