RxJS mergeAll

核心概念：mergeAll 将高阶 Observable（发出内部 Observable 的 Observable）展平，并行订阅所有内部 Observable。

函数签名

mergeAll(concurrent?: number): OperatorFunction, T>

说明

常与 map 组合实现类似 mergeMap 的效果。concurrent 参数限制同时活跃的 inner Observable 数量。

所有 inner Observable 的值交错发出到输出流。

参数

参数说明
concurrent（可选）最大并发 inner 数量，默认 Infinity

代码示例

import { of, timer } from 'rxjs'; import { map, mergeAll } from 'rxjs/operators'; of(0, 1, 2).pipe( map(i => timer(i * 100)), mergeAll() ).subscribe(console.log); // 0, 1, 2（交错发出）
