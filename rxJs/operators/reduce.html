RxJS reduce

核心概念：reduce 对 Observable 发出的所有值应用累加器函数，仅在源完成时发出最终的累积结果（类似 Array.reduce）。

签名

reduce<T, R>(accumulator: (acc: R, value: T, index: number) => R, seed: R): OperatorFunction<T, R> reduce<T, R>(accumulator: (acc: R | T, value: T, index: number) => R): OperatorFunction<T, R>

参量

  • accumulator：累加器函数 (acc, value, index) => newAcc
  • seed（可选）：初始累积值；不提供时以第一个值为初始值

返回值

返回 OperatorFunction，源 complete 时发出单个累积结果。

描述

与 scan 的区别：reduce 只在完成时发出一次最终结果，scan 每次累积都发出中间值。

例子

import { of } from 'rxjs'; import { reduce } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( reduce((acc, val) => acc + val, 0) ).subscribe(console.log); // 输出: 15 of(1, 2, 3).pipe( reduce((acc, val) => acc + val) ).subscribe(console.log); // 输出: 6（无 seed，以第一个值 1 为初始 acc）
