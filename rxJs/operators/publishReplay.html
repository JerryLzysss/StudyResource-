RxJS publishReplay

核心概念：publishReplay 使用 ReplaySubject 进行 multicast，新订阅者会收到指定数量的历史值重放。

签名

publishReplay<T>(bufferSize?: number, windowTime?: number, scheduler?: SchedulerLike): MonoTypeOperatorFunction<T>

参量

  • bufferSize（可选）：重放的最大值数量，默认 Infinity
  • windowTime（可选）：重放的时间窗口（毫秒）
  • scheduler（可选）：Scheduler

返回值

返回 ConnectableObservable，等价于 multicast(new ReplaySubject(bufferSize, windowTime))。

描述

适用于晚到订阅者需要获取历史数据的场景。配合 refCount() 即 shareReplay。

例子

import { interval } from 'rxjs'; import { publishReplay, take } from 'rxjs/operators'; const shared = interval(500).pipe( take(5), publishReplay(2) ); shared.connect(); setTimeout(() => shared.subscribe(v => console.log('晚到:', v)), 2000);
