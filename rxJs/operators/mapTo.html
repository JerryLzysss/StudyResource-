RxJS mapTo

核心概念：mapTo 将源 Observable 的每个值映射为同一个常量值，忽略原始值内容。

函数签名

mapTo(value: R): OperatorFunction

说明

map 的简化版，当你不关心源值、只需发出固定信号时使用。

例如将任意点击事件映射为 true，或将多个触发源统一映射为同一命令。

参数

参数说明
value每个源值映射到的常量

代码示例

import { of } from 'rxjs'; import { mapTo } from 'rxjs/operators'; of(1, 2, 3) .pipe(mapTo('OK')) .subscribe(console.log); // 输出: OK, OK, OK
