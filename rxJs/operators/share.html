RxJS share

核心概念：share 将冷 Observable 转为热 Observable，多个订阅者共享同一底层订阅，等价于 publish().refCount()。

签名

share<T>(): MonoTypeOperatorFunction<T>

返回值

返回 MonoTypeOperatorFunction，第一个订阅者触发订阅，最后一个取消时断开。

描述

share 是最常用的多播运算符，适用于 HTTP 请求、事件流等需要共享的场景。不缓存历史值（与 shareReplay 不同）。

例子

import { interval } from 'rxjs'; import { share, tap } from 'rxjs/operators'; const shared = interval(1000).pipe( tap(v => console.log('副作用:', v)), share() ); shared.subscribe(v => console.log('A:', v)); shared.subscribe(v => console.log('B:', v)); // 两个订阅者共享同一 interval，副作用只执行一次
