RxJS elementAt

核心概念：elementAt 从源 Observable 中按发射顺序取出指定索引位置的单个值并发出，然后完成。

签名

elementAt<T>( index: number, defaultValue?: T ): MonoTypeOperatorFunction<T>

描述

elementAt 返回一个 Observable，它在源 Observable 中指定 index 位置（从 0 开始）发出该项目，然后完成。如果 index 超出范围且提供了 defaultValue，则发出默认值；否则抛出 ArgumentOutOfRangeError。

index 小于 0 时也会抛出 ArgumentOutOfRangeError。

例子

import { fromEvent } from 'rxjs'; import { elementAt } from 'rxjs/operators'; const clicks = fromEvent(document, 'click'); clicks.pipe(elementAt(2)).subscribe(x => console.log(x)); // 结果: // 第 1 次点击 = 无输出 // 第 2 次点击 = 无输出 // 第 3 次点击 = MouseEvent 对象输出到控制台
