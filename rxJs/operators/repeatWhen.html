RxJS repeatWhen

核心概念：repeatWhen 在源 complete 时，根据 notifier Observable 决定是否重新订阅源，实现自定义重复逻辑。

签名

repeatWhen<T>(notifier: (notifications: Observable<void>) => Observable<any>): MonoTypeOperatorFunction<T>

参量

  • notifier：接收 complete 通知的 Observable，返回决定是否重复的 Observable

返回值

返回 MonoTypeOperatorFunction，notifier 发出值时重新订阅源。

描述

repeatWhen 提供比 repeat 更灵活的控制，如延迟重复、条件重复、有限次数等。

例子

import { of, timer } from 'rxjs'; import { repeatWhen, delayWhen, take } from 'rxjs/operators'; of(1, 2, 3).pipe( repeatWhen(notifications => notifications.pipe( delayWhen(() => timer(1000)), take(2) ) ) ).subscribe(console.log); // 每次 complete 后延迟 1s 重复，最多 2 次
