RxJS windowCount

核心概念：windowCount 将源 Observable 的值按固定数量分割为多个窗口 Observable，每个窗口包含 windowSize 个值。

签名

windowCount<T>( windowSize: number, startWindowEvery?: number ): OperatorFunction<T, Observable<T>>

参量

  • windowSize：每个窗口的值数量
  • startWindowEvery（可选）：每隔多少个值开启新窗口（滑动窗口）

返回值

返回 OperatorFunction，发出固定大小的窗口 Observable。

例子

import { of } from 'rxjs'; import { windowCount, mergeMap, toArray } from 'rxjs/operators'; of(1, 2, 3, 4, 5, 6, 7).pipe( windowCount(3), mergeMap(w => w.pipe(toArray())) ).subscribe(console.log); // [1,2,3], [4,5,6], [7]
