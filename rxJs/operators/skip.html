RxJS skip

核心概念：skip 跳过源 Observable 发出的前 n 个值，从第 n+1 个值开始向下游传递。

签名

skip<T>(count: number): MonoTypeOperatorFunction<T>

参量

  • count：要跳过的值的数量

返回值

返回 MonoTypeOperatorFunction，跳过前 count 个值。

例子

import { of } from 'rxjs'; import { skip } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( skip(2) ).subscribe(console.log); // 输出: 3, 4, 5
