RxJS windowWhen

核心概念：windowWhen 在每个窗口开始时调用 closingSelector 获取关闭窗口的 Observable，该 Observable 发出值时关闭当前窗口并开启新窗口。

签名

windowWhen<T>( closingSelector: () => ObservableInput<any> ): OperatorFunction<T, Observable<T>>

参量

  • closingSelector：返回关闭当前窗口的 Observable 的函数

返回值

返回 OperatorFunction，由 closingSelector 控制窗口关闭时机。

描述

windowWhen 在第一个值到达时开启第一个窗口，closingSelector 返回的 Observable 发出值时关闭窗口并开启新窗口。

例子

import { interval, timer } from 'rxjs'; import { windowWhen, take, mergeMap, toArray } from 'rxjs/operators'; interval(100).pipe( windowWhen(() => timer(800)), take(3), mergeMap(w => w.pipe(toArray())) ).subscribe(console.log);
