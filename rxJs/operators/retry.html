RxJS retry

核心概念：retry 在源 Observable 发生 error 时重新订阅，可指定最大重试次数，常用于网络请求失败重试。

签名

retry<T>(count?: number): MonoTypeOperatorFunction<T>

参量

  • count（可选）：最大重试次数，默认 Infinity

返回值

返回 MonoTypeOperatorFunction，error 时重新订阅源。

描述

retry 只在 error 时重试，complete 不会触发重试。超过重试次数后将 error 传递给下游。

例子

import { defer, throwError, of } from 'rxjs'; import { retry, map } from 'rxjs/operators'; let attempt = 0; defer(() => { attempt++; if (attempt < 3) return throwError(() => new Error('失败')); return of('成功'); }).pipe( retry(2) ).subscribe(console.log); // 第 3 次成功，输出: '成功'
