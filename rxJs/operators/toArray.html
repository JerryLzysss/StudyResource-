RxJS toArray

核心概念：toArray 收集源 Observable 发出的所有值，在源 complete 时以数组形式一次性发出。

签名

toArray<T>(): OperatorFunction<T, T[]>

返回值

返回 OperatorFunction，源 complete 时发出包含所有值的数组。

描述

toArray 不会发出中间值，只在 complete 时发出单个数组。空源发出空数组 []。

例子

import { of } from 'rxjs'; import { toArray } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( toArray() ).subscribe(console.log); // 输出: [1, 2, 3, 4, 5]
