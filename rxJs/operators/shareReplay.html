RxJS shareReplay

核心概念：shareReplay 共享订阅并缓存指定数量的历史值，新订阅者会收到重放的值，等价于 publishReplay().refCount()。

签名

shareReplay<T>(bufferSize?: number, windowTime?: number, scheduler?: SchedulerLike): MonoTypeOperatorFunction<T>

参量

  • bufferSize（可选）：缓存值数量，默认 Infinity
  • windowTime（可选）：缓存时间窗口（毫秒）

返回值

返回 MonoTypeOperatorFunction，共享订阅并重放历史值。

描述

常用于 HTTP 请求缓存、状态共享等场景，避免重复请求且让晚到订阅者获取已有数据。

例子

import { interval } from 'rxjs'; import { shareReplay, take } from 'rxjs/operators'; const shared = interval(500).pipe( take(4), shareReplay(2) ); shared.subscribe(v => console.log('A:', v)); setTimeout(() => shared.subscribe(v => console.log('B:', v)), 1500);
