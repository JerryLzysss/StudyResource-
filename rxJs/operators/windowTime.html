RxJS windowTime

核心概念：windowTime 将源 Observable 的值按固定时间间隔分割为多个窗口 Observable。

签名

windowTime<T>( windowTimeSpan: number, windowCreationInterval?: number, maxWindowSize?: number, scheduler?: SchedulerLike ): OperatorFunction<T, Observable<T>>

参量

  • windowTimeSpan：每个窗口的持续时间（毫秒）
  • windowCreationInterval（可选）：创建新窗口的间隔（滑动窗口）
  • maxWindowSize（可选）：每个窗口的最大值数量

返回值

返回 OperatorFunction，按时间分窗。

例子

import { interval } from 'rxjs'; import { windowTime, take, mergeMap, toArray } from 'rxjs/operators'; interval(100).pipe( windowTime(500), take(3), mergeMap(w => w.pipe(toArray())) ).subscribe(console.log); // 每 500ms 一个窗口
