RxJS publishLast

核心概念：publishLast 使用 AsyncSubject 进行 multicast，仅在源完成时向所有订阅者发出最后一个值。

签名

publishLast<T>(): MonoTypeOperatorFunction<T>

返回值

返回 ConnectableObservable，等价于 multicast(new AsyncSubject())。

描述

与 AsyncSubject 行为一致：完成前订阅的观察者会在 complete 时收到最后一个值；完成后再订阅也会立即收到最后一个值。

例子

import { of } from 'rxjs'; import { publishLast } from 'rxjs/operators'; const shared = of(1, 2, 3).pipe(publishLast()); shared.subscribe(v => console.log('A:', v)); shared.subscribe(v => console.log('B:', v)); shared.connect(); // 源完成后输出: // A: 3 // B: 3
