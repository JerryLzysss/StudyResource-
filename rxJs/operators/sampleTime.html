RxJS sampleTime

核心概念：sampleTime 每隔指定时间间隔采样源 Observable 的最近一个值并发出，常用于降低高频数据流的更新频率。

签名

sampleTime<T>(period: number, scheduler?: SchedulerLike): MonoTypeOperatorFunction<T>

参量

  • period：采样间隔（毫秒）
  • scheduler（可选）：Scheduler

返回值

返回 MonoTypeOperatorFunction，按固定间隔采样。

描述

与 throttleTime 不同：sampleTime 按固定节奏采样最新值；throttleTime 在指定时间内只发出第一个值。

例子

import { interval } from 'rxjs'; import { sampleTime } from 'rxjs/operators'; interval(100).pipe( sampleTime(500) ).subscribe(console.log); // 每 500ms 发出 interval 的当前值
