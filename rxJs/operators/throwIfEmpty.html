RxJS throwIfEmpty

核心概念：throwIfEmpty 在源 Observable complete 且未发出任何值时，抛出 EmptyError；否则正常透传。

签名

throwIfEmpty<T>(errorFactory?: () => any): MonoTypeOperatorFunction<T>

参量

  • errorFactory（可选）：自定义错误工厂，默认抛出 EmptyError

返回值

返回 MonoTypeOperatorFunction，空序列时 error。

描述

常用于确保 Observable 至少有一个值，如查询结果不能为空。

例子

import { EMPTY, of } from 'rxjs'; import { throwIfEmpty } from 'rxjs/operators'; EMPTY.pipe( throwIfEmpty(() => new Error('没有数据')) ).subscribe( v => console.log(v), err => console.log(err.message) // '没有数据' ); of(1).pipe(throwIfEmpty()).subscribe(console.log); // 1
