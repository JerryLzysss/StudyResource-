RxJS groupBy

核心概念：groupBy 将源 Observable 的值按 keySelector 分组，每个分组作为独立的 GroupedObservable 发出。

函数签名

groupBy(keySelector, elementSelector?, durationSelector?): OperatorFunction>

说明

对每个源值计算分组键，相同键的值归入同一 GroupedObservable。

常用于按类别、ID 等维度拆分数据流，再分别处理各组。

参数

参数说明
keySelector从每个值提取分组键的函数
elementSelector（可选）转换分组内元素的函数
durationSelector（可选）决定分组何时关闭

代码示例

import { of } from 'rxjs'; import { groupBy, mergeMap } from 'rxjs/operators'; of( { type: 'A', val: 1 }, { type: 'B', val: 2 }, { type: 'A', val: 3 }, { type: 'B', val: 4 } ).pipe( groupBy(p => p.type), mergeMap(group => group.pipe( mapTo(group.key + ': ' + group.key) )) ).subscribe(console.log);
