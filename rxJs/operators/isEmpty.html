RxJS isEmpty

核心概念：isEmpty 在源 Observable 完成时发出 true（若无任何 next 值）或 false（若至少有一个值）。

函数签名

isEmpty(): OperatorFunction

说明

源 Observable 发出任何 next 值后，isEmpty 会记住并等待 complete。

完成时：若从未收到 next 则发出 true，否则发出 false。常用于判断流是否为空。

代码示例

import { of, EMPTY } from 'rxjs'; import { isEmpty } from 'rxjs/operators'; EMPTY.pipe(isEmpty()).subscribe(console.log); // true of(1, 2, 3).pipe(isEmpty()).subscribe(console.log); // false
