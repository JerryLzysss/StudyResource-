RxJS sequenceEqual

核心概念：sequenceEqual 比较两个 Observable 是否按相同顺序发出相同的值，两个源都 complete 时发出 true 或 false。

签名

sequenceEqual<T>(compareTo: Observable<T>, comparator?: (a: T, b: T) => boolean): OperatorFunction<T, boolean>

参量

  • compareTo：用于比较的另一个 Observable
  • comparator（可选）：自定义比较函数，默认 ===

返回值

返回 OperatorFunction，发出单个 boolean 值表示序列是否相等。

描述

若两个序列长度不同、任一值不相等、或任一源 error，则发出 false。两个源都 complete 且所有值匹配时发出 true。

例子

import { of } from 'rxjs'; import { sequenceEqual } from 'rxjs/operators'; of(1, 2, 3).pipe( sequenceEqual(of(1, 2, 3)) ).subscribe(console.log); // 输出: true of(1, 2, 3).pipe( sequenceEqual(of(1, 2, 4)) ).subscribe(console.log); // 输出: false
