RxJS retryWhen

核心概念：retryWhen 在源 error 时，根据 notifier Observable 决定是否重新订阅，实现自定义重试策略（如指数退避）。

签名

retryWhen<T>(notifier: (errors: Observable<any>) => Observable<any>): MonoTypeOperatorFunction<T>

参量

  • notifier：接收 error 通知的 Observable，返回决定是否重试的 Observable

返回值

返回 MonoTypeOperatorFunction，notifier 发出值时重新订阅源。

描述

retryWhen 比 retry 更灵活，可实现延迟重试、指数退避、条件重试等高级策略。

例子

import { defer, throwError, of, timer } from 'rxjs'; import { retryWhen, delayWhen, scan, take } from 'rxjs/operators'; let attempt = 0; defer(() => { attempt++; if (attempt < 3) return throwError(() => new Error('失败')); return of('成功'); }).pipe( retryWhen(errors => errors.pipe( scan((count, err) => count + 1, 0), delayWhen(count => timer(count * 500)), take(2) ) ) ).subscribe(console.log);
