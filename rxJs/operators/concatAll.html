RxJS concatAll

核心概念：concatAll 用于处理高阶 Observable（Observable 的 Observable），按顺序订阅每个内部 Observable，等上一个完成后才订阅下一个，并合并它们发出的值。

签名

concatAll(): OperatorFunction<ObservableInput<T>, T>

描述

当源 Observable 发出一个内部 Observable 时，concatAll 会订阅该内部 Observable。只有当当前内部 Observable 完成后，才会订阅源发出的下一个内部 Observable。所有内部 Observable 发出的值会按顺序合并输出。

等价于 mergeAll(1)，即并发数为 1 的 mergeAll。常用于将 Observable 流按顺序展开，避免并发订阅。

例子

import { of } from 'rxjs'; import { concatAll } from 'rxjs/operators'; const obs1 = of(1, 2); const obs2 = of(3, 4); const source = of(obs1, obs2); source.pipe(concatAll()).subscribe(x => console.log(x)); // 输出: 1, 2, 3, 4
