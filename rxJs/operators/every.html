RxJS every

核心概念：every 检测源 Observable 发出的所有值是否都满足给定条件，在源完成时发出 true 或 false。

签名

every<T>( predicate: (value: T, index: number) => boolean, thisArg?: any ): OperatorFunction<T, boolean>

描述

every 类似数组的 Array.every 方法。它订阅源 Observable，对每个值应用 predicate 函数。如果所有值都满足条件，在源完成时发出 true；只要有一个值不满足，立即发出 false 并完成。

如果源 Observable 为空（没有发出任何值），every 会发出 true。

例子

import { of } from 'rxjs'; import { every } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( every(val => val % 2 === 0) ).subscribe(x => console.log(x)); // 输出: false of(2, 4, 6, 8, 10).pipe( every(val => val % 2 === 0) ).subscribe(x => console.log(x)); // 输出: true
