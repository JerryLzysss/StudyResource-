RxJS switchAll

核心概念：switchAll 订阅 Observable 的 Observable，切换到最新内部 Observable 并取消之前的，等价于 switchMap(identity)。

签名

switchAll<O extends ObservableInput<any>>(): OperatorFunction<O, ObservedValueOf<O>>

返回值

返回 OperatorFunction，展平高阶 Observable 并切换到最新内部 Observable。

描述

当源发出新的内部 Observable 时，取消对之前内部 Observable 的订阅。适用于源已经是 Observable 的 Observable 的场景。

例子

import { from, of, interval } from 'rxjs'; import { switchAll, map, take } from 'rxjs/operators'; const source = from(['a', 'b', 'c']).pipe( map(x => interval(500).pipe( map(i => x + i), take(3) )) ); source.pipe(switchAll()).subscribe(console.log); // 只输出 'c' 开头的值（之前的被取消）
