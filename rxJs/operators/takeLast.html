RxJS takeLast

核心概念：takeLast 缓存源发出的值，在源 complete 时只发出最后 n 个值。

签名

takeLast<T>(count: number): MonoTypeOperatorFunction<T>

参量

  • count：要取的最后 n 个值的数量

返回值

返回 MonoTypeOperatorFunction，源 complete 时发出最后 count 个值。

描述

与 take 相反：take 取开头 n 个，takeLast 取末尾 n 个。需要源 complete 才会发出值。

例子

import { of } from 'rxjs'; import { takeLast } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( takeLast(2) ).subscribe(console.log); // 输出: 4, 5
