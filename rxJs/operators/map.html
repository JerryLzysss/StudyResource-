RxJS map

核心概念：map 对源 Observable 发出的每个值应用 project 函数，将结果作为新值发出，类似 Array.prototype.map。

函数签名

map(project: (value: T, index: number) => R): OperatorFunction

说明

最常用的转换运算符之一。不改变发射次数，只转换每个值的类型或内容。

可链式组合多个 map 实现复杂转换。

参数

参数说明
project转换函数，接收值和索引，返回新值

代码示例

import { of } from 'rxjs'; import { map } from 'rxjs/operators'; of(1, 2, 3) .pipe(map(x => x * 10)) .subscribe(console.log); // 输出: 10, 20, 30
