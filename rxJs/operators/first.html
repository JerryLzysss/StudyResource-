RxJS first

核心概念：first 发出源 Observable 的第一个值（或满足条件的第一个值），然后完成；若无值则报错或发出默认值。

函数签名

first(predicate?, defaultValue?): OperatorFunction

说明

仅发出第一个匹配的值后立即 complete。可传入 predicate 筛选第一个满足条件的值。

若源为空且未提供 defaultValue，会抛出 EmptyError。

参数

参数说明
predicate（可选）筛选条件
defaultValue（可选）源为空时的默认值

代码示例

import { of } from 'rxjs'; import { first } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe(first(x => x > 2)) .subscribe(console.log); // 输出: 3
