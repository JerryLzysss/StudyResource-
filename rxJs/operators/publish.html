RxJS publish

核心概念：publish 是 multicast(new Subject()) 的简写，将冷 Observable 转为 ConnectableObservable，多个订阅者共享同一底层订阅。

签名

publish<T>(): MonoTypeOperatorFunction<T> publish<T>(selector: (shared: Observable<T>) => Observable<any>): OperatorFunction<T, any>

参量

  • selector（可选）：接收共享 Observable 并返回新 Observable 的函数

返回值

无 selector 时返回 ConnectableObservable；有 selector 时自动 connect 并返回 selector 结果。

描述

publish 常用于需要手动控制 connect/disconnect 时机的场景。配合 refCount() 可自动管理连接。

例子

import { interval } from 'rxjs'; import { publish, tap } from 'rxjs/operators'; const shared = interval(1000).pipe( tap(v => console.log('源:', v)), publish() ); shared.subscribe(v => console.log('A:', v)); shared.subscribe(v => console.log('B:', v)); shared.connect();
