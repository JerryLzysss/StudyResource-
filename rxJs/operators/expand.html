RxJS expand

核心概念：expand 递归地应用给定函数，将每个源值映射为新的 Observable 并合并输出，实现自引用展开。

签名

expand<T, O extends ObservableInput<T>>( project: (value: T, index: number) => O, concurrent?: number, scheduler?: SchedulerLike ): OperatorFunction<T, T>

描述

expand 对每个发出的值调用 project 函数，返回一个新的 Observable。expand 会订阅这些 Observable 并将其值合并到输出流中，同时对每个新值再次调用 project，形成递归展开。

常用于实现树形结构遍历、分页加载（每次请求返回下一页 URL）等需要递归获取数据的场景。使用 take 等运算符限制展开深度，避免无限递归。

例子

import { of } from 'rxjs'; import { expand, take } from 'rxjs/operators'; of(2).pipe( expand(val => of(val + 1)), take(5) ).subscribe(x => console.log(x)); // 输出: 2, 3, 4, 5, 6
