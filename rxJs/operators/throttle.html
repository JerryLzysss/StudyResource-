RxJS throttle

核心概念：throttle 在 duration Observable 发出值时，发出源的最新值并忽略期间的值，由 notifier 控制节流节奏。

签名

throttle<T>( durationSelector: (value: T) => ObservableInput<any>, config?: ThrottleConfig ): MonoTypeOperatorFunction<T>

参量

  • durationSelector：为每个值返回 duration Observable 的函数
  • config（可选）：{ leading?, trailing? } 控制首尾行为

返回值

返回 MonoTypeOperatorFunction，按 duration 节奏节流。

描述

throttle 比 throttleTime 更灵活，duration 可以依赖源值。throttleTime 是固定时间间隔的特例。

例子

import { interval, timer } from 'rxjs'; import { throttle } from 'rxjs/operators'; interval(200).pipe( throttle(() => timer(800)) ).subscribe(console.log); // 每 800ms 发出 interval 的当前值
