RxJS mergeScan

核心概念：mergeScan 类似 scan，但累加器返回 Observable，并将所有 inner Observable 合并到输出流。

函数签名

mergeScan(accumulator, seed, concurrent?): OperatorFunction

说明

对每个源值，accumulator 接收当前累积值和新值，返回 Observable。

所有返回的 inner Observable 被 merge 到输出。适合需要异步累加的场景，如分页加载。

参数

参数说明
accumulator(acc, value, index) => ObservableInput
seed初始累积值
concurrent（可选）最大并发数，默认 Infinity

代码示例

import { of, timer } from 'rxjs'; import { mergeScan, mapTo } from 'rxjs/operators'; of(1, 2, 3).pipe( mergeScan((acc, val) => timer(50).pipe(mapTo(acc + val)), 0) ).subscribe(console.log); // 1, 3, 6
