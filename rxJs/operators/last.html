RxJS last

核心概念：last 发出源 Observable 的最后一个值（或满足条件的最后一个值），然后完成。

函数签名

last(predicate?, defaultValue?): OperatorFunction

说明

等待源 complete 后发出最后一个值。可传入 predicate 筛选最后一个满足条件的值。

若源为空且未提供 defaultValue，会抛出 EmptyError。与 first 相对。

参数

参数说明
predicate（可选）筛选条件
defaultValue（可选）源为空时的默认值

代码示例

import { of } from 'rxjs'; import { last } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe(last(x => x < 4)) .subscribe(console.log); // 输出: 3
