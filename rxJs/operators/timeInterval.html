RxJS timeInterval

核心概念：timeInterval 将每个源值包装为 TimeInterval 对象，包含 value 和自上一个值以来经过的 interval（毫秒）。

签名

timeInterval<T>(scheduler?: SchedulerLike): OperatorFunction<T, TimeInterval<T>>

参量

  • scheduler（可选）：Scheduler，默认 asyncScheduler

返回值

返回 OperatorFunction，发出 { value, interval } 对象。

描述

第一个值的 interval 是从订阅到第一个值的时间。常用于测量事件间隔、性能监控。

例子

import { interval } from 'rxjs'; import { timeInterval, take } from 'rxjs/operators'; interval(500).pipe( timeInterval(), take(3) ).subscribe(console.log); // { value: 0, interval: ~500 } // { value: 1, interval: ~500 } // { value: 2, interval: ~500 }
