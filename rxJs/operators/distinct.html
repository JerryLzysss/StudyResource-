RxJS distinct

核心概念：distinct 返回一个 Observable，只发出与之前所有值都不同的值（全局去重）。

签名

distinct<T, K>( keySelector?: (value: T) => K, flushes?: Observable<any> ): MonoTypeOperatorFunction<T>

描述

distinct 使用内部 Set 记录已出现过的值。每当源发出一个值，如果该值（或通过 keySelector 投影后的值）之前未出现过，则发出该值；否则忽略。

与 distinctUntilChanged 不同，distinct 是全局去重——即使中间隔了其他值，重复的值也不会再次发出。可选的 flushes 参数用于清空内部 Set，缓解长期运行时的内存问题。

例子

import { of } from 'rxjs'; import { distinct } from 'rxjs/operators'; of(1, 1, 2, 2, 2, 1, 2, 3, 4, 3, 2, 1).pipe( distinct() ).subscribe(x => console.log(x)); // 输出: 1, 2, 3, 4 // 使用 keySelector of( { age: 4, name: 'Foo' }, { age: 7, name: 'Bar' }, { age: 5, name: 'Foo' } ).pipe( distinct(p => p.name) ).subscribe(x => console.log(x)); // 输出: { age: 4, name: 'Foo' }, { age: 7, name: 'Bar' }
