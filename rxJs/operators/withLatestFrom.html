RxJS withLatestFrom

核心概念：withLatestFrom 当源 Observable 发出值时，将其与其他 Observable 的最新值组合成数组发出。其他源的变化不会触发输出。

签名

withLatestFrom<T, R>(...inputs: ObservableInput<any>[]): OperatorFunction<T, [T, ...R[]]> withLatestFrom<T, R>(inputs: ObservableInput<any>[], project?: (...values: any[]) => R): OperatorFunction<T, R>

参量

  • inputs：要组合最新值的其他 Observable
  • project（可选）：组合函数

返回值

返回 OperatorFunction，源发出时组合其他源的最新值。

描述

与 combineLatest 不同：withLatestFrom 只在主源发出时触发，其他源仅提供最新值。常用于按钮点击时获取最新表单值。

例子

import { fromEvent, interval, BehaviorSubject } from 'rxjs'; import { withLatestFrom } from 'rxjs/operators'; const clicks = fromEvent(document, 'click'); const timer = interval(1000); const latest = new BehaviorSubject('初始'); clicks.pipe( withLatestFrom(timer, latest) ).subscribe(([click, t, l]) => console.log(click, t, l));
