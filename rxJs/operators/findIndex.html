RxJS findIndex

核心概念：findIndex 发出源 Observable 中第一个满足谓词条件的值的索引，然后完成；若无匹配则发出 -1。

函数签名

findIndex(predicate: (value: T, index: number) => boolean): Observable

说明

类似 Array.prototype.findIndex。找到匹配项后发出其索引（从 0 开始）并 complete。

若未找到，发出 -1 后完成。

参数

参数说明
predicate查找条件

代码示例

import { of } from 'rxjs'; import { findIndex } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe(findIndex(x => x > 3)) .subscribe(console.log); // 输出: 3
