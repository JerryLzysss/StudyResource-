RxJS skipUntil

核心概念：skipUntil 跳过源的值，直到 notifier Observable 发出第一个值，之后才开始向下游传递源的值。

签名

skipUntil<T>(notifier: Observable<any>): MonoTypeOperatorFunction<T>

参量

  • notifier：触发开始传递的 Observable

返回值

返回 MonoTypeOperatorFunction，notifier 发出值后开始传递源的值。

例子

import { interval, timer } from 'rxjs'; import { skipUntil } from 'rxjs/operators'; interval(300).pipe( skipUntil(timer(1500)) ).subscribe(console.log); // 1.5 秒前跳过，之后开始发出 interval 的值
