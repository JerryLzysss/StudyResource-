RxJS zipAll

核心概念：zipAll 订阅 Observable 的 Observable，将所有内部 Observable 拉链式组合，一对一配对发出数组，等价于 mergeMap(source => source).pipe(toArray()) 的 zip 变体。

签名

zipAll<T>(): OperatorFunction<ObservableInput<T>, T[]>

返回值

返回 OperatorFunction，将高阶 Observable 中的内部 Observable 拉链组合。

描述

zipAll 用于处理 emit Observable 的 Observable，将所有内部流按索引拉链配对。当任一内部 Observable complete 时，zipAll 也 complete。

例子

import { of } from 'rxjs'; import { zipAll } from 'rxjs/operators'; of( of(1, 2, 3), of('a', 'b', 'c') ).pipe( zipAll() ).subscribe(console.log); // [1, 'a'] // [2, 'b'] // [3, 'c']
