RxJS scan

核心概念：scan 对源发出的每个值应用累加器，并发出每次累积的中间结果，类似 reduce 但会发出所有中间值。

签名

scan<T, R>(accumulator: (acc: R, value: T, index: number) => R, seed: R): OperatorFunction<T, R>

参量

  • accumulator：累加器函数
  • seed：初始累积值

返回值

返回 OperatorFunction，每次累积都发出中间结果。

描述

scan 常用于状态管理、累计求和、构建历史记录等。与 reduce 的区别是 scan 发出所有中间值。

例子

import { of } from 'rxjs'; import { scan } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( scan((acc, val) => acc + val, 0) ).subscribe(console.log); // 输出: 1, 3, 6, 10, 15
