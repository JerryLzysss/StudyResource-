RxJS mergeMapTo

核心概念：mergeMapTo 将每个源值映射为同一个内部 Observable 并合并，忽略源值本身。

函数签名

mergeMapTo(innerObservable, resultSelector?, concurrent?): OperatorFunction

说明

mergeMap 的简化版，inner Observable 不依赖源值。

适合每个触发事件都执行相同异步操作的场景。

参数

参数说明
innerObservable每个源值触发的内部 Observable
concurrent（可选）最大并发数

代码示例

import { of, timer } from 'rxjs'; import { mergeMapTo } from 'rxjs/operators'; of(1, 2, 3).pipe( mergeMapTo(timer(100).pipe(mapTo('done'))) ).subscribe(console.log); // done, done, done
