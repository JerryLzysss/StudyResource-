RxJS bufferWhen

核心概念：将过往的值收集到数组中。开始收集时调用函数返回 Observable，该 Observable 告知何时关闭缓冲区并重新开始收集。

签名

bufferWhen<T>( closingSelector: () => ObservableInput<any> ): OperatorFunction<T, T[]>

参量

参量说明
closingSelector不接受参数，返回通知缓冲区关闭的 Observable

返回值

返回 OperatorFunction<T, T[]>：缓冲数组的 Observable。

描述

立马开启缓冲区，然后当 closingSelector 函数返回的 Observable 发出数据的时候关闭缓冲区。当关闭缓冲区的时候，会立马开启新的缓冲区，并不断重复此过程。

例子

发出每个随机秒（1-5 秒）数内的最新点击事件数组

import { fromEvent, interval } from 'rxjs'; import { bufferWhen } from 'rxjs/operators'; const clicks = fromEvent(document, 'click'); const buffered = clicks.pipe( bufferWhen(() => interval(1000 + Math.random() * 4000)) ); buffered.subscribe(x => console.log(x));
