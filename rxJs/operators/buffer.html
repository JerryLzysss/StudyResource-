RxJS buffer

核心概念：将过往的值收集到一个数组中，并且仅当另一个 Observable 发出通知时才发出此数组。

签名

buffer<T>( closingNotifier: ObservableInput<any> ): OperatorFunction<T, T[]>

参量

参量说明
closingNotifier该 Observable 向输出 Observable 发出信号以通知发出缓冲区

返回值

返回 OperatorFunction<T, T[]>：数组缓冲的 Observable。

描述

缓冲源 Observable 的值直到 closingNotifier 发出。将 Observable 发出的值缓冲起来直到 closingNotifier 发出数据，在这个时候在输出 Observable 上发出该缓冲区的值并且内部开启一个新的缓冲区，等待下一个 closingNotifier 的发送。

例子

每次点击发出 interval Observable 最新缓冲的数组

import { fromEvent, interval } from 'rxjs'; import { buffer } from 'rxjs/operators'; const clicks = fromEvent(document, 'click'); const intervalEvents = interval(1000); const buffered = intervalEvents.pipe(buffer(clicks)); buffered.subscribe(x => console.log(x));
