RxJS mergeMap

核心概念：mergeMap（又称 flatMap）将每个源值映射为内部 Observable 并并行合并发出，适合并发请求场景。

函数签名

mergeMap(project, resultSelector?, concurrent?): OperatorFunction

说明

对每个值调用 project 得到 inner Observable，订阅所有 inner 并将它们的值合并到输出。

与 switchMap（取消前一个）和 concatMap（串行）不同，mergeMap 并行处理所有 inner。

参数

参数说明
project映射为 ObservableInput 的函数
concurrent（可选）最大并发数

代码示例

import { of, timer } from 'rxjs'; import { mergeMap } from 'rxjs/operators'; of('请求1', '请求2', '请求3').pipe( mergeMap(req => timer(100).pipe(mapTo(req + ' 完成'))) ).subscribe(console.log);
