RxJS throttleTime

核心概念：throttleTime 在指定时间间隔内只发出第一个值（leading）或最后一个值（trailing），常用于滚动、拖拽等高频事件优化。

签名

throttleTime<T>( duration: number, scheduler?: SchedulerLike, config?: ThrottleConfig ): MonoTypeOperatorFunction<T>

参量

  • duration：节流时间间隔（毫秒）
  • config（可选）：{ leading: true, trailing: false } 默认 leading

返回值

返回 MonoTypeOperatorFunction，限制发出频率。

例子

import { fromEvent } from 'rxjs'; import { throttleTime } from 'rxjs/operators'; fromEvent(document, 'scroll').pipe( throttleTime(200) ).subscribe(() => console.log('滚动')); // 200ms 内最多触发一次
