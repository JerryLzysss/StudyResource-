RxJS repeat

核心概念：repeat 在源 Observable 完成（complete）后重新订阅并重复发出值，可指定重复次数。

签名

repeat<T>(count?: number): MonoTypeOperatorFunction<T>

参量

  • count（可选）：重复次数，默认 Infinity（无限重复）

返回值

返回 MonoTypeOperatorFunction，源 complete 后重新订阅。

描述

repeat 只在 complete 时重复，遇到 error 不会重复。与 retry 相反：retry 在 error 时重试。

例子

import { of } from 'rxjs'; import { repeat } from 'rxjs/operators'; of(1, 2, 3).pipe( repeat(2) ).subscribe(console.log); // 输出: 1, 2, 3, 1, 2, 3（重复 2 次，共 3 轮）
