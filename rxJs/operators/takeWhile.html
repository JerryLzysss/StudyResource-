RxJS takeWhile

核心概念：takeWhile 持续发出满足 predicate 的值，一旦 predicate 返回 false 就 complete（inclusive 模式可包含该值）。

签名

takeWhile<T>( predicate: (value: T, index: number) => boolean, inclusive?: boolean ): MonoTypeOperatorFunction<T>

参量

  • predicate：返回 true 时继续发出值
  • inclusive（可选）：true 时包含第一个不满足条件的值

返回值

返回 MonoTypeOperatorFunction，predicate 返回 false 时 complete。

例子

import { of } from 'rxjs'; import { takeWhile } from 'rxjs/operators'; of(1, 2, 3, 4, 5).pipe( takeWhile(v => v <= 3) ).subscribe(console.log); // 输出: 1, 2, 3
