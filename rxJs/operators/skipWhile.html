RxJS skipWhile

核心概念：skipWhile 跳过源中满足 predicate 的值，一旦 predicate 返回 false，之后所有值都会传递（不再检查 predicate）。

签名

skipWhile<T>(predicate: (value: T, index: number) => boolean): MonoTypeOperatorFunction<T>

参量

  • predicate：返回 true 时跳过该值

返回值

返回 MonoTypeOperatorFunction，跳过开头连续满足条件的值。

例子

import { of } from 'rxjs'; import { skipWhile } from 'rxjs/operators'; of(1, 2, 3, 4, 5, 6).pipe( skipWhile(v => v < 4) ).subscribe(console.log); // 输出: 4, 5, 6
