RxJS publishBehavior

核心概念：publishBehavior 使用 BehaviorSubject 进行 multicast，新订阅者会立即收到当前最新值。

签名

publishBehavior<T>(initialValue: T): MonoTypeOperatorFunction<T>

参量

  • initialValue：BehaviorSubject 的初始值

返回值

返回 ConnectableObservable，等价于 multicast(new BehaviorSubject(initialValue))。

描述

适用于需要共享状态且晚到订阅者需要获取最新值的场景，如当前用户信息、主题设置等。

例子

import { interval } from 'rxjs'; import { publishBehavior } from 'rxjs/operators'; const shared = interval(1000).pipe( publishBehavior(0) ); shared.subscribe(v => console.log('A:', v)); setTimeout(() => shared.subscribe(v => console.log('B:', v)), 2500); shared.connect();
