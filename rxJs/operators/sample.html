RxJS sample

核心概念：sample 在 notifier Observable 每次发出值时，采样源 Observable 的最近一个值并发出。

签名

sample<T>(notifier: Observable<any>): MonoTypeOperatorFunction<T>

参量

  • notifier：采样触发 Observable，每次发出值时采样源的最新值

返回值

返回 MonoTypeOperatorFunction，按 notifier 节奏采样源的值。

描述

sample 与 sampleTime 类似，但采样时机由 notifier 控制而非固定时间间隔。若采样时没有新值，则不发出。

例子

import { interval, timer } from 'rxjs'; import { sample } from 'rxjs/operators'; interval(100).pipe( sample(timer(0, 500)) ).subscribe(console.log); // 每 500ms 采样一次 interval 的当前值
