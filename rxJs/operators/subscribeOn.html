RxJS subscribeOn

核心概念：subscribeOn 使用指定的 Scheduler 调度对源 Observable 的订阅（subscribe 调用），控制订阅发生的执行上下文。

签名

subscribeOn<T>(scheduler: SchedulerLike, delay?: number): MonoTypeOperatorFunction<T>

参量

  • scheduler：SchedulerLike，如 asyncScheduler
  • delay（可选）：延迟订阅的时间（毫秒）

返回值

返回 MonoTypeOperatorFunction，订阅会在指定 Scheduler 上异步执行。

描述

subscribeOn 只影响订阅时机，不影响通知投递（通知投递由 observeOn 控制）。常用于将阻塞操作移到后台线程。

例子

import { of, asyncScheduler } from 'rxjs'; import { subscribeOn } from 'rxjs/operators'; of(1, 2, 3).pipe( subscribeOn(asyncScheduler) ).subscribe(v => console.log(v)); // 订阅在 asyncScheduler 上异步执行
