RxJS windowToggle

核心概念：windowToggle 根据 openings 和 closingSelector 动态开启和关闭窗口，openings 发出值时开启新窗口，closingSelector 返回的 Observable complete 时关闭窗口。

签名

windowToggle<T, O>( openings: ObservableInput<O>, closingSelector: (openValue: O) => ObservableInput<any> ): OperatorFunction<T, Observable<T>>

参量

  • openings：发出值时开启新窗口的 Observable
  • closingSelector：为每个 open 值返回关闭窗口的 Observable

返回值

返回 OperatorFunction，动态开关窗口。

描述

windowToggle 提供最灵活的窗口控制，适用于开始/结束事件驱动的分组场景。

例子

import { interval, timer } from 'rxjs'; import { windowToggle, take, mergeMap, toArray } from 'rxjs/operators'; interval(100).pipe( windowToggle( timer(0, 1500), () => timer(800) ), take(2), mergeMap(w => w.pipe(toArray())) ).subscribe(console.log);
