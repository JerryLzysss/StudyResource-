RxJS AjaxRequest

核心概念：AjaxRequest 是 RxJS 用于发起 HTTP 请求的配置对象，包含 url、method、headers、body 等参数。传入 ajax() 时使用。

接口

interface AjaxRequest { url: string body?: any method: string async: boolean headers: Readonly<Record<string, any>> timeout: number user?: string password?: string crossDomain: boolean withCredentials: boolean responseType: XMLHttpRequestResponseType }

描述

该对象包含 RxJS 发起 HTTP 请求时使用的各项值。在 AjaxError 实例中可通过 request 属性访问。

属性

属性类型说明
urlstring请求的 URL
bodyanyHTTP 请求体（可选）
methodstringHTTP 方法（GET、POST 等）
asyncboolean是否异步请求
headersReadonly<Record<string, any>>请求头
timeoutnumber超时时间（毫秒），仅 async 为 true 时生效
userstringHTTP 基本认证用户名（可选）
passwordstringHTTP 基本认证密码（可选）
crossDomainboolean是否为跨域请求
withCredentialsboolean跨域请求是否携带凭证
responseTypeXMLHttpRequestResponseType响应类型

代码示例

import { ajax } from 'rxjs/ajax'; const request: AjaxRequest = { url: 'https://api.github.com/users?per_page=5', method: 'GET', headers: { 'Accept': 'application/json' } }; ajax(request).subscribe(res => console.log(res.response));
