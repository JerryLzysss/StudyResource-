RxJS AjaxResponse

核心概念：AjaxResponse 是标准化的 AJAX 响应对象，由 ajax() 成功时发出，包含 status、response、xhr 等完整响应信息。

类定义

class AjaxResponse { constructor(originalEvent: Event, xhr: XMLHttpRequest, request: AjaxRequest) status: number response: any responseText: string responseType: XMLHttpRequestResponseType originalEvent: Event xhr: XMLHttpRequest request: AjaxRequest }

构造函数

constructor(originalEvent: Event, xhr: XMLHttpRequest, request: AjaxRequest)

参数类型
originalEventEvent
xhrXMLHttpRequest
requestAjaxRequest

属性

属性类型说明
statusnumberHTTP 状态码
responseany解析后的响应体
responseTextstring原始响应文本
responseTypeXMLHttpRequestResponseType响应类型
originalEventEvent原始 load 事件
xhrXMLHttpRequest底层 XMLHttpRequest
requestAjaxRequest对应的请求配置

代码示例

import { ajax } from 'rxjs/ajax'; ajax('https://api.github.com/users?per_page=3').subscribe(res => { console.log('status:', res.status); console.log('response:', res.response); console.log('responseType:', res.responseType); console.log('request.url:', res.request.url); });
