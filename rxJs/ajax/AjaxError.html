RxJS AjaxError

核心概念：AjaxError 是 ajax 请求失败时抛出的错误类型，继承自 Error，包含 XMLHttpRequest、请求配置和响应详情。

接口

interface AjaxError extends Error { xhr: XMLHttpRequest request: AjaxRequest status: number responseType: XMLHttpRequestResponseType response: any }

子类型

  • AjaxTimeoutError

属性

属性类型说明
xhrXMLHttpRequest底层 XMLHttpRequest 对象
requestAjaxRequest发起请求时使用的配置对象
statusnumberHTTP 状态码
responseTypeXMLHttpRequestResponseType响应类型
responseany响应体内容

代码示例

import { ajax, AjaxError } from 'rxjs/ajax'; import { catchError } from 'rxjs/operators'; import { of } from 'rxjs'; ajax('https://api.github.com/404').pipe( catchError((err: AjaxError) => { console.log('名称:', err.name); // AjaxError console.log('状态:', err.status); // 404 console.log('请求 URL:', err.request.url); console.log('响应:', err.response); return of(err); }) ).subscribe();
