RxJS WebSocketSubject

核心概念：WebSocketSubject 继承自 AnonymousSubject，是可通过 WebSocket 与服务器通信的 Subject，支持双向消息传递与多路复用。

类定义

class WebSocketSubject<T> extends AnonymousSubject { constructor( urlConfigOrSource: string | WebSocketSubjectConfig<T> | Observable<T>, destination?: Observer<T> ) _output: Subject<T> lift<R>(operator: Operator<T, R>): WebSocketSubject<R> multiplex( subMsg: () => any, unsubMsg: () => any, messageFilter: (value: T) => boolean ) _subscribe(subscriber: Subscriber<T>): Subscription unsubscribe() }

构造函数

constructor(urlConfigOrSource, destination?)

参数类型
urlConfigOrSourcestring | WebSocketSubjectConfig | Observable
destination可选，默认 undefined，类型 Observer

属性

属性类型描述
_outputSubject<T>内部输出 Subject

方法

方法说明
lift(operator) 应用操作符，返回 WebSocketSubject<R>
multiplex(subMsg, unsubMsg, messageFilter) 创建 Observable：订阅时通过 subMsg 向服务器发送订阅消息；数据到达后用 messageFilter 筛选；拆卸时通过 unsubMsg 发送退订消息。消息仍由配置中的序列化器处理（默认 JSON）
_subscribe(subscriber) 订阅 WebSocketSubject，返回 Subscription
unsubscribe() 取消订阅，无参数

multiplex 参量

参数说明
subMsg生成订阅消息的函数，发送到服务器
unsubMsg生成退订消息的函数，在拆卸时发送到服务器
messageFilter从服务器消息中筛选适当数据的谓词
