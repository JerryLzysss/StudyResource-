RxJS WebSocketSubjectConfig

核心概念：WebSocketSubjectConfig 是普通配置对象，用于自定义 webSocket 的行为，包括序列化/反序列化策略及连接生命周期观察者。

接口定义

interface WebSocketSubjectConfig<T> { url: string protocol?: string | Array<string> resultSelector?: (e: MessageEvent) => T serializer?: (value: T) => WebSocketMessage deserializer?: (e: MessageEvent) => T openObserver?: NextObserver<Event> closeObserver?: NextObserver<CloseEvent> closingObserver?: NextObserver<void> WebSocketCtor?: { new(url: string, protocols?: string | string[]): WebSocket } binaryType?: 'blob' | 'arraybuffer' }

描述

提供灵活性给 webSocket。它定义了一组属性以在套接字生命周期的特定时刻提供自定义行为。连接打开后可用 openObserver，连接关闭时用 closeObserver；通过 deserializer 可自定义从服务器到达数据的反序列化策略。默认 deserializer 对每条服务器消息应用 JSON.parse。

属性

属性类型描述
urlstring要连接的套接字服务器 URL
protocolstring | Array<string>用于连接的协议
resultSelector(e: MessageEvent) => T结果选择器
serializer(value: T) => WebSocketMessage发送前创建消息的序列化器，默认为 JSON.stringify
deserializer(e: MessageEvent) => T反序列化器，默认为 JSON.parse
openObserverNextObserver<Event>监视 WebSocket 打开事件
closeObserverNextObserver<CloseEvent>监视 WebSocket 关闭事件
closingObserverNextObserver<void>监视因取消订阅等原因即将发生的关闭
WebSocketCtorWebSocket 构造函数用于 Node 环境或测试时模拟 WebSocket
binaryType'blob' | 'arraybuffer'二进制数据类型

示例

deserializer — 自定义反序列化

import { webSocket } from 'rxjs/webSocket'; const wsSubject = webSocket({ url: 'ws://localhost:8081', deserializer: ({ data }) => data }); wsSubject.subscribe(console.log); // 服务器发送: ws.send("This is a msg from the server") // 输出: This is a msg from the server

serializer — 自定义序列化

import { webSocket } from 'rxjs/webSocket'; const wsSubject = webSocket({ url: 'ws://localhost:8081', serializer: msg => JSON.stringify({ channel: 'webDevelopment', msg: msg }) }); wsSubject.subscribe(() => wsSubject.next('msg to the server')); // 发送: {"channel":"webDevelopment","msg":"msg to the server"}

closeObserver — 自定义关闭处理

import { webSocket } from 'rxjs/webSocket'; const wsSubject = webSocket({ url: 'ws://localhost:8081', closeObserver: { next(closeEvent) { const customError = { code: 6666, reason: '自定义错误原因' }; console.log(`code: ${customError.code}, reason: ${customError.reason}`); } } }); // 输出: code: 6666, reason: Custom evil reason

openObserver — 连接成功通知

import { webSocket } from 'rxjs/webSocket'; const wsSubject = webSocket({ url: 'ws://localhost:8081', openObserver: { next: () => console.log('连接正常') } }); // 输出: connection ok
