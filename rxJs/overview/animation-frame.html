RxJS 动画帧调度器

核心概念：animationFrameScheduler 是 RxJS 提供的调度器之一，它使用浏览器的 requestAnimationFrame API 来调度任务。任务将在下一次浏览器重绘之前执行，非常适合创建流畅的浏览器动画。

什么是调度程序？

调度程序控制何时开始订阅以及何时传递通知。它由三个部分组成：

  • 数据结构：知道如何根据优先级或其他条件存储和排队任务
  • 执行上下文：表示执行任务的位置和时间（例如立即执行、setTimeout、requestAnimationFrame 等）
  • 虚拟时钟：通过 now() 方法提供"时间"的概念
调度程序使您可以定义 Observable 在哪个执行上下文中将通知传递给其 Observer。

animationFrameScheduler 的特点

特性 说明
执行时机 在下一次浏览器内容重绘之前执行
API 基础 基于 requestAnimationFrame
刷新率同步 自动与显示器刷新率同步（通常 60fps）
节能特性 页面不可见时自动暂停，节省 CPU/GPU 资源
适用场景 流畅的浏览器动画、视觉效果

与其他调度器的对比

调度器 执行机制 适用场景
null / undefined 同步执行 恒定时间操作
queueScheduler 当前事件帧队列 迭代操作
asapScheduler 微任务队列 (Promise) 异步转换
asyncScheduler setTimeout/setInterval 基于时间的操作
animationFrameScheduler requestAnimationFrame 流畅动画

基本使用示例

1. 使用 observeOn 控制通知传递

import { Observable, animationFrameScheduler } from 'rxjs'; import { observeOn } from 'rxjs/operators'; const observable = new Observable((observer) => { observer.next(1); observer.next(2); observer.next(3); observer.complete(); }).pipe( observeOn(animationFrameScheduler) ); console.log('订阅前'); observable.subscribe({ next(x) { console.log('收到值 ' + x); }, complete() { console.log('完成'); } }); console.log('订阅后'); // 输出顺序: // 订阅前 // 订阅后 // 收到值 1 // 收到值 2 // 收到值 3 // 完成

2. 使用 subscribeOn 控制订阅时机

import { of, animationFrameScheduler } from 'rxjs'; import { subscribeOn } from 'rxjs/operators'; const observable = of(1, 2, 3).pipe( subscribeOn(animationFrameScheduler) ); console.log('订阅前'); observable.subscribe(x => console.log('值: ' + x)); console.log('订阅后'); // 输出顺序: // 订阅前 // 订阅后 // 值: 1 // 值: 2 // 值: 3

3. 创建动画序列

import { interval, animationFrameScheduler } from 'rxjs'; import { take, map } from 'rxjs/operators'; // 使用 animationFrameScheduler 创建流畅动画 const animation$ = interval(0, animationFrameScheduler).pipe( take(60), // 运行 60 帧（约 1 秒） map(frame => frame / 60) // 进度从 0 到 1 ); const element = document.getElementById('box'); animation$.subscribe(progress => { // 使用 requestAnimationFrame 的流畅动画 element.style.transform = `translateX(${progress * 300}px)`; });

动画实战示例

平滑滚动动画

import { interval, animationFrameScheduler } from 'rxjs'; import { takeWhile, map } from 'rxjs/operators'; // 平滑滚动到页面顶部 function smoothScrollToTop() { const startY = window.scrollY; const duration = 500; // 500ms const startTime = performance.now(); return interval(0, animationFrameScheduler).pipe( map(() => performance.now() - startTime), takeWhile(elapsed => elapsed < duration), map(elapsed => { const progress = elapsed / duration; // 使用缓动函数 const easeProgress = 1 - Math.pow(1 - progress, 3); return startY * (1 - easeProgress); }) ).subscribe(y => window.scrollTo(0, y)); }

进度条动画

import { of, animationFrameScheduler } from 'rxjs'; import { delayWhen, observeOn, map } from 'rxjs/operators'; // 模拟文件上传进度 function uploadWithProgress(files) { let uploaded = 0; return of(...files).pipe( delayWhen(() => interval(100 + Math.random() * 400)), map(file => { uploaded++; return { file, progress: (uploaded / files.length) * 100 }; }) ); } // 使用 animationFrameScheduler 更新进度条 uploadWithProgress(files).pipe( observeOn(animationFrameScheduler) ).subscribe(({ progress }) => { progressBar.style.width = `${progress}%`; });
