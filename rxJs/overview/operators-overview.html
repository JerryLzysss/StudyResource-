RxJS 运算符

核心概念：运算符是 Observable 类型上的方法，用于处理数据流。它们可以帮助你创建、转换、过滤、组合和操作数据。RxJS 提供了丰富的运算符库，让异步编程变得更加简单。

什么是运算符？

运算符是 RxJS 的核心功能，它们是用来处理 Observable 的函数。使用 pipe() 方法可以将多个运算符组合在一起：

import { of } from 'rxjs'; import { map, filter, take } from 'rxjs/operators'; of(1, 2, 3, 4, 5) .pipe( filter(x => x > 2), map(x => x * 2), take(2) ) .subscribe(console.log); // 输出: 6, 8

运算符分类

1. 创建类运算符 (Creation Operators)

运算符 说明 用途
of() 将参数转换为 Observable 创建发出固定值的 Observable
from() 从数组、Promise 等创建 Observable 转换其他数据类型为 Observable
interval() 定期发出递增的数字 定时任务、轮询
timer() 延迟后发出值或周期性发出 延迟执行、定时器
fromEvent() 将事件转换为 Observable 处理 DOM 事件
ajax() 创建 HTTP 请求的 Observable 发起网络请求

2. 转换类运算符 (Transformation Operators)

运算符 说明 用途
map() 对每个值进行转换 数据转换、格式化
mapTo() 将所有值映射为固定值 信号转换
mergeMap() 展平内部 Observable（并行） 处理并发请求
switchMap() 切换到新的 Observable，取消之前的 搜索、自动完成
concatMap() 展平内部 Observable（串行） 顺序执行任务
scan() 累加器，类似 reduce 但发出中间结果 累计计算、状态管理

3. 过滤类运算符 (Filtering Operators)

运算符 说明 用途
filter() 只通过符合条件的值 数据筛选
take() 只取前 n 个值 限制数据量
takeUntil() 直到另一个 Observable 发出值时停止 取消订阅控制
debounceTime() 防抖，延迟发出值 搜索输入优化
throttleTime() 节流，限制发出频率 滚动事件优化
distinctUntilChanged() 过滤连续重复的值 去重

4. 组合类运算符 (Combination Operators)

运算符 说明 用途
merge() 合并多个 Observable 并行处理多个流
concat() 顺序连接多个 Observable 串行执行任务
combineLatest() 组合最新值 多数据源同步
zip() 拉链式组合，一对一配对 同步多个流
forkJoin() 等待所有 Observable 完成后发出最终值 并行请求等待

5. 错误处理类运算符 (Error Handling Operators)

运算符 说明 用途
catchError() 捕获错误并返回新的 Observable 错误恢复
retry() 发生错误时重试 网络请求重试
retryWhen() 自定义重试逻辑 高级重试策略
finalize() Observable 结束时执行清理 资源清理

6. 工具类运算符 (Utility Operators)

运算符 说明 用途
tap() 执行副作用，不改变数据 调试、日志记录
delay() 延迟发出值 延迟执行
timeout() 超时处理 请求超时控制
shareReplay() 共享订阅并缓存 多订阅者共享

常用代码示例

map - 数据转换
import { from } from 'rxjs'; import { map } from 'rxjs/operators'; from([1, 2, 3, 4, 5]) .pipe(map(x => x * 10)) .subscribe(console.log); // 输出: 10, 20, 30, 40, 50
filter - 数据过滤
import { from } from 'rxjs'; import { filter } from 'rxjs/operators'; from([1, 2, 3, 4, 5, 6]) .pipe(filter(x => x % 2 === 0)) .subscribe(console.log); // 输出: 2, 4, 6 (只输出偶数)
switchMap - 切换 Observable
import { of } from 'rxjs'; import { switchMap, delay } from 'rxjs/operators'; of('userId: 123') .pipe( switchMap(id => of(`Data for ${id}`).pipe(delay(1000))) ) .subscribe(console.log); // 常用于 HTTP 请求切换
debounceTime - 防抖
import { fromEvent } from 'rxjs'; import { debounceTime, map } from 'rxjs/operators'; fromEvent(inputElement, 'input') .pipe( debounceTime(300), map(e => e.target.value) ) .subscribe(value => console.log('搜索:', value)); // 用户停止输入 300ms 后才触发
combineLatest - 组合多个流
import { combineLatest, BehaviorSubject } from 'rxjs'; const weight = new BehaviorSubject(70); const height = new BehaviorSubject(1.75); combineLatest([weight, height]) .pipe(map(([w, h]) => w / (h * h))) .subscribe(bmi => console.log('BMI:', bmi));
catchError - 错误处理
import { throwError, of } from 'rxjs'; import { catchError } from 'rxjs/operators'; throwError(() => new Error('出错啦')) .pipe( catchError(err => { console.log('捕获错误:', err.message); return of('默认值'); }) ) .subscribe(console.log); // 输出: 捕获错误: 出错啦 // 输出: 默认值
