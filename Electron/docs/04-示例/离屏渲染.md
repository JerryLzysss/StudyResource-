# 离屏渲染

> 官方文档：[Offscreen Rendering | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/offscreen-rendering)

在不显示窗口的情况下渲染网页内容，适用于截图、视频录制等场景。

```js
const win = new BrowserWindow({
  webPreferences: { offscreen: true }
})
win.webContents.on('paint', (event, dirty, image) => {
  // 处理渲染帧
})
```
