# 多线程

> 官方文档：[Multithreading | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/multithreading)

在 Electron 中使用 Web Workers 或 Node.js Worker Threads 实现多线程。

```js
// 渲染器中使用 Web Worker
const worker = new Worker('worker.js')
worker.postMessage('start')
worker.onmessage = (e) => console.log(e.data)
```
