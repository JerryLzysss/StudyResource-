# 原生文件拖放

> 官方文档：[Native File Drag & Drop | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/native-file-drag-drop)

实现从 Electron 应用向操作系统拖放文件。

```js
// 主进程
ipcMain.on('ondragstart', (event, filePath) => {
  event.sender.startDrag({
    file: filePath,
    icon: '/path/to/icon.png'
  })
})
```
