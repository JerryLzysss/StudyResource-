# 应用菜单

> 官方文档：[Application Menu | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/application-menu)

使用 `Menu` 模块创建应用菜单栏。

```js
const { Menu } = require('electron')
const template = [
  { label: '文件', submenu: [{ role: 'quit' }] }
]
Menu.setApplicationMenu(Menu.buildFromTemplate(template))
```
