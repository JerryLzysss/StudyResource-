# 上下文菜单

> 官方文档：[Context Menu | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/context-menu)

使用 `Menu.buildFromTemplate` 创建右键上下文菜单。

```js
const { Menu } = require('electron')

win.webContents.on('context-menu', (event, params) => {
  const menu = Menu.buildFromTemplate([
    { label: '复制', role: 'copy' },
    { label: '粘贴', role: 'paste' }
  ])
  menu.popup()
})
```
