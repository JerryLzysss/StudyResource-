# 在线/离线检测

> 官方文档：[Online/Offline Detection | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/online-offline-events)

检测网络连接状态。

```js
// 渲染器
window.addEventListener('online', () => console.log('已连接'))
window.addEventListener('offline', () => console.log('已断开'))

// 主进程
const { net } = require('electron')
net.isOnline() // true/false
```
