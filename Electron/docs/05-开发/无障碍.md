# 无障碍

> 官方文档：[Accessibility | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/accessibility)

Electron 应用会自动继承 Chromium 的无障碍功能。可以手动启用无障碍功能进行测试。

```js
const { app } = require('electron')
app.setAccessibilitySupportEnabled(true)
```

在 macOS 上，可以使用 `VoiceOver` 测试；Windows 上使用 `Narrator`；Linux 上使用 `Orca`。
