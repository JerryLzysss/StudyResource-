# 样板与 CLI

> 官方文档：[Boilerplates and CLIs | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/boilerplates-and-clis)

## 常用工具

| 工具 | 说明 |
|------|------|
| [Electron Forge](https://www.electronforge.io/) | 官方推荐的打包和分发工具 |
| [electron-builder](https://www.electron.build/) | 第三方打包工具，功能丰富 |
| [electron-react-boilerplate](https://github.com/electron-react-boilerplate/electron-react-boilerplate) | React + Electron 样板项目 |
| [create-electron-app](https://www.electronforge.io/) | Forge 官方脚手架命令 |

## 快速开始

```sh
npm init electron-app@latest my-app
```
