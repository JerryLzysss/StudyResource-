# 自动化测试

> 官方文档：[Automated Testing | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/automated-testing)

## 测试框架

| 框架 | 说明 |
|------|------|
| [Playwright](https://playwright.dev/) | 推荐，支持 Electron |
| [Spectron](https://www.electronjs.org/spectron) | 已弃用 |
| WebdriverIO | 支持 Electron 驱动 |

## Playwright 示例

```js
const { _electron: electron } = require('playwright')

const app = await electron.launch({ args: ['main.js'] })
const window = await app.firstWindow()
await window.click('button')
await app.close()
```

## 测试类型

- **单元测试**：测试独立函数和模块
- **集成测试**：测试 IPC 通信和进程交互
- **端到端测试**：测试完整用户流程
