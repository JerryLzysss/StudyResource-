# 应用调试

> 官方文档：[Application Debugging | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/application-debugging)

## 调试工具概览

| 工具 | 用途 |
|------|------|
| Chrome DevTools | 调试渲染器进程 |
| `--inspect` | 调试主进程 |
| VS Code | 集成调试环境 |
| `contentTracing` | 性能追踪 |

## 常用调试技巧

1. 使用 `console.log` 在主进程和渲染器中输出日志
2. 使用 `win.webContents.openDevTools()` 打开开发者工具
3. 监听 `webContents` 的 `did-fail-load` 事件排查加载问题
4. 使用 `app.commandLine.appendSwitch('enable-logging')` 启用详细日志
