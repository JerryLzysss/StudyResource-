# 调试主进程

> 官方文档：[Debugging Main Process | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/debugging-main-process)

## 使用 --inspect 标志

```sh
electron --inspect=5858 .
```

然后在 Chrome 中访问 `chrome://inspect` 连接调试器。

## 使用 --inspect-brk

在首行代码处暂停：

```sh
electron --inspect-brk=5858 .
```
