# 更新

> 官方文档：[Updates | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/updates)

## autoUpdater 模块

Electron 内置 `autoUpdater` 模块，支持 macOS 和 Windows 的自动更新。

```js
const { autoUpdater } = require('electron')

autoUpdater.setFeedURL({ url: 'https://your-server.com/updates' })
autoUpdater.checkForUpdates()

autoUpdater.on('update-downloaded', () => {
  autoUpdater.quitAndInstall()
})
```

## 更新服务

| 服务 | 说明 |
|------|------|
| [update.electronjs.org](https://update.electronjs.org/) | Electron 团队提供的免费更新服务 |
| Electron Forge | 集成 GitHub Releases 发布 |
| electron-builder | 内置多种更新提供商 |

## Squirrel.Windows

Windows 上推荐使用 Squirrel.Windows 作为更新框架，Electron Forge 默认使用 `@electron-forge/maker-squirrel`。
