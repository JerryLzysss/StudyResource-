# Windows Store

> 官方文档：[Windows Store | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/windows-store-guide)

## 概述

将 Electron 应用发布到 Microsoft Store 需要：

1. Microsoft Partner Center 账号
2. 将应用打包为 APPX/MSIX 格式
3. 使用 Windows SDK 工具转换

## 工具

使用 `electron-windows-store` 或 electron-builder 的 Windows Store 目标进行打包。
