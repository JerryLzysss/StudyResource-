# 应用分发

> 官方文档：[Application Distribution | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/application-distribution)

## 概述

Electron 应用分发需要将应用代码与 Electron 运行时打包在一起。推荐使用 [Electron Forge](../Electron-Forge概述.md) 或 electron-builder。

## 打包工具

| 工具 | 命令 |
|------|------|
| Electron Forge | `npm run make` |
| @electron/packager | `npx electron-packager .` |
| electron-builder | `npm run build` |

## 输出格式

| 平台 | 格式 |
|------|------|
| Windows | `.exe` (Squirrel/NSIS)、`.msi` |
| macOS | `.dmg`、`.pkg`、`.app` |
| Linux | `.deb`、`.rpm`、`.AppImage`、`.snap` |
