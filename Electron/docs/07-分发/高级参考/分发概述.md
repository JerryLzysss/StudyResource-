# 分发概述

> 官方文档：[Distribution Overview | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/distribution-overview)

## 分发流程

1. **打包**：将应用代码与 Electron 二进制文件结合
2. **签名**：对应用进行代码签名（macOS/Windows）
3. **生成安装包**：创建各平台的安装程序
4. **发布**：上传到分发平台
5. **更新**：配置自动更新机制
