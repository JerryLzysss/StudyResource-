# 代码签名

> 官方文档：[Code Signing | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/code-signing)

## 概述

代码签名用于证明桌面应用程序是由已知来源创建的。未签名的应用在 Windows 和 macOS 上可能难以下载或启动。

## 平台差异

| 平台 | 签名时机 | 证书类型 |
|------|----------|----------|
| macOS | 打包时 | Apple Developer ID |
| Windows | 分发文件签名 | Authenticode 证书 |

## Forge 配置示例

```js
// macOS
makers: [{
  name: '@electron-forge/maker-dmg',
  config: {
    sign: { identity: 'Developer ID Application: ...' }
  }
}]

// Windows
makers: [{
  name: '@electron-forge/maker-squirrel',
  config: {
    certificateFile: './cert.pfx',
    certificatePassword: process.env.CERTIFICATE_PASSWORD
  }
}]
```
