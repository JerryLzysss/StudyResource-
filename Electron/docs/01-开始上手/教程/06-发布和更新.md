# 发布和更新

> 官方文档：[发布和更新 | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/tutorial-6-publishing-updating)

这是 Electron 教程的第 6 部分。

## 概述

完成应用打包后，需要将应用发布给用户，并实现自动更新功能。Electron 提供了 `autoUpdater` 模块来支持应用更新。

## 发布到 GitHub Releases

Electron Forge 支持将应用发布到 GitHub Releases：

```sh
npm run publish
```

需要在 `forge.config.js` 中配置 publishers：

```js
module.exports = {
  publishers: [
    {
      name: '@electron-forge/publisher-github',
      config: {
        repository: {
          owner: 'your-username',
          name: 'your-repo'
        }
      }
    }
  ]
}
```

## 自动更新

使用 `autoUpdater` 模块实现自动更新：

```js
const { autoUpdater } = require('electron')

autoUpdater.setFeedURL({
  url: 'https://your-update-server.com/updates'
})

autoUpdater.on('update-downloaded', () => {
  autoUpdater.quitAndInstall()
})

app.whenReady().then(() => {
  autoUpdater.checkForUpdates()
})
```

## 更新方案

| 方案 | 说明 |
|------|------|
| `autoUpdater` | Electron 内置，适用于 macOS 和 Windows |
| `update.electronjs.org` | Electron 团队提供的免费更新服务 |
| Electron Forge | 集成发布和更新流程 |
| electron-builder | 第三方打包工具，内置更新支持 |

更多详情请参阅 [高级参考 - 更新](../../07-分发/高级参考/更新.md)。
