# 简介

> 官方文档：[简介 | Electron](https://www.electronjs.org/zh/docs/latest/)

## Electron 是什么？

Electron 是一个使用 JavaScript、HTML 和 CSS 构建桌面应用程序的框架。通过将 [Chromium](https://www.chromium.org/) 和 [Node.js](https://nodejs.org/) 嵌入到其二进制文件中，Electron 允许你保持一个 JavaScript 代码库，并创建在 Windows、macOS 和 Linux 上运行的跨平台应用——不需要本地开发经验。

## 入门指南

建议从[教程](./教程/01-基本要求.md)开始，该教程将指导你完成开发 Electron 应用程序并将其分发给用户的过程。[示例](../04-示例/示例预览.md)和 [API 文档](https://www.electronjs.org/zh/docs/latest/api/app)也是浏览和发现新事物的好地方。

## Electron Fiddle 运行实例

[Electron Fiddle](https://www.electronjs.org/fiddle) 是由 Electron 开发并由其维护者支持的沙盒程序。强烈建议将其作为一个学习工具来安装，以便在开发过程中对 Electron 的 API 进行实验或对特性进行原型化。

Fiddle 已经完美地集成到官方帮助文档之中。当你浏览教程中的例子，你会发现有个「Open In Electron Fiddle」按钮在代码示例中。
