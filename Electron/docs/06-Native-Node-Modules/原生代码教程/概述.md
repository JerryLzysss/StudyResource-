# 原生代码与 Electron 概述

> 官方文档：[Native Code and Electron | Electron](https://www.electronjs.org/zh/docs/latest/tutorial/native-code-and-electron)

## 概述

Electron 允许你编写原生代码（C++、Swift、Rust 等）来扩展应用功能。原生代码通过 Node.js 原生插件系统集成。

## 平台教程

| 平台 | 教程 |
|------|------|
| Windows | [C++ Win32](./C++-Win32.md) |
| macOS | [Swift macOS](./Swift-macOS.md) |
| Linux | [C++ Linux](./C++-Linux.md) |

## 使用场景

- 调用系统 API
- 高性能计算
- 集成现有 C/C++ 库
- 硬件设备访问
