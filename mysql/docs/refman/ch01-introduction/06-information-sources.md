## 1.5 MySQL信息源

    本节列出了您可能会觉得有用的其他信息来源，例如 MySQL 网站、邮件列表、用户论坛和 Internet 中继聊天。

- MySQL 网站
- MySQL 论坛上的 MySQL 社区支持
- 企业版

### MySQL 网站

      MySQL 文档的主要网站是
       参考手册、MySQL 连接器等提供在线和可下载的文档格式。

[MySQL 开发人员在MySQL 服务器博客](http://mysqlserverteam.com/)
      中提供有关新功能和即将推出的功能的信息。

### MySQL 论坛上的 MySQL 社区支持

[http://forums.mysql.com](http://forums.mysql.com)上
      的论坛是一个重要的社区资源。许多论坛可用，分为以下一般类别：

- 移民
- MySQL 用法
- MySQL 连接器
- 编程语言
- 工具
- 第三方应用程序
- 存储引擎
- MySQL技术
- SQL 标准
- 商业

### 企业版

      Oracle 以 MySQL Enterprise 的形式提供技术支持。对于依赖 MySQL DBMS 进行关键业务生产应用程序的组织，MySQL Enterprise 是一种商业订阅产品，其中包括：

- MySQL企业服务器
- MySQL 企业监视器
- 每月快速更新和季度服务包
- MySQL 知识库
- 24x7 技术和咨询支持

      MySQL Enterprise 提供多个层级，让您可以灵活地选择最符合您需求的服务级别。有关详细信息，请参阅
      [MySQL 企业版](https://www.mysql.com/products/enterprise/)。
