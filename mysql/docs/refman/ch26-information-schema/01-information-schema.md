# 第 26 章 INFORMATION_SCHEMA 表

    `INFORMATION_SCHEMA`提供对数据库
    元数据的访问，有关 MySQL 服务器的信息，例如数据库或表的名称、列的数据类型或访问权限。有时用于此信息的其他术语是
    数据字典和
    系统目录。
