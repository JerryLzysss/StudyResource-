# MySQL 8.0 参考手册（第 1–28 章 · 按小节拆分）

> 结构：`docs/refman/chXX-*/NN-小节.md` + `sql/refman/chXX-*/NN-小节.sql`

## 使用方式

1. 打开某章索引，例如 [`ch10-charset/00-index.md`](ch10-charset/00-index.md)
2. 进入单个小节文档阅读
3. 执行对应小节 SQL，例如：

```bash
mysql -u root -p < sql/refman/ch10-charset/01-charset.sql
```
