## 12.21 窗口函数

      MySQL 支持窗口函数，对于查询中的每一行，使用与该行相关的行执行计算。以下部分讨论如何使用窗口函数，包括对`OVER`and
      `WINDOW`子句的描述。第一部分提供非聚合窗口函数的描述。有关聚合窗口函数的说明，请参阅
      第 12.20.1 节，“聚合函数说明”。

      有关优化和窗口函数的信息，请参阅
      第 8.2.1.21 节，“窗口函数优化”。
