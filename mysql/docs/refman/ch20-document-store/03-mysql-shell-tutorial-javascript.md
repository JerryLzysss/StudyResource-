## 20.3 JavaScript 快速入门指南：用于文档存储的 MySQL Shell

    本快速入门指南提供了开始使用 MySQL Shell 交互地构建文档存储应用程序原型的说明。该指南包括以下主题：

- 介绍 MySQL 功能、MySQL Shell 和 world_x示例架构。
- 管理集合和文档的操作。
- 管理关系表的操作。
- 适用于表中文档的操作。

    要遵循此快速入门指南，您需要一个安装了 X 插件的 MySQL 服务器（8.0 中的默认插件）和用作客户端的 MySQL Shell。MySQL Shell 8.0
    提供了有关 MySQL Shell 的更深入的信息。使用 X DevAPI 访问文档存储，MySQL Shell 在 JavaScript 和 Python 中提供此 API。
