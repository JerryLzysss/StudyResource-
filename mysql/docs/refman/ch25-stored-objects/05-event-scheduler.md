## 25.4 使用事件调度器

MySQL Event Scheduler管理事件
    的
    调度和执行，即根据计划运行的任务。以下讨论涵盖事件调度程序，分为以下部分：

- 第 25.4.1 节，“事件调度程序概述”，提供了 MySQL 事件的介绍和概念概述。
- 第 25.4.3 节，“事件语法”，讨论了用于创建、更改和删除 MySQL 事件的 SQL 语句。
- 第 25.4.4 节，“事件元数据”，展示了如何获取有关事件的信息以及 MySQL 服务器如何存储这些信息。
- 第 25.4.6 节，“事件调度程序和 MySQL 特权”，讨论了处理事件所需的特权以及事件在执行时对特权的影响。

    存储例程需要系统数据库`events`中的数据字典表。`mysql`该表是在 MySQL 8.0 安装过程中创建的。如果您是从早期版本升级到 MySQL 8.0，请务必执行升级程序以确保您的系统数据库是最新的。请参阅
    第 2.11 节，“升级 MySQL”。
