## 11.3 字符串数据类型

      字符串数据类型为`CHAR`,
       `VARCHAR`,
       `BINARY`,
       `VARBINARY`,
       `BLOB`,
       `TEXT`,
      `ENUM`和
      `SET`。

      有关字符串数据类型的存储要求的信息，请参阅第 11.7 节，“数据类型存储要求”。

      有关对字符串值进行操作的函数的描述，请参阅
      第 12.8 节，“字符串函数和运算符”。
