# 08 WHERE 子句

> 独立模块。对应 SQL：[`sql/tutorial/08-where.sql`](../../sql/tutorial/08-where.sql)

---

按条件筛选，也可用于 `UPDATE` / `DELETE`。

| 运算符 | 含义 |
|--------|------|
| `=` | 等于 |
| `<>` / `!=` | 不等于 |
| `>` / `<` / `>=` / `<=` | 大小比较 |
| `AND` / `OR` | 逻辑与 / 或 |
| `BETWEEN` | 范围 |
| `IN` | 集合 |
