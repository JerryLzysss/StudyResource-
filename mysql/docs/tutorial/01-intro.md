## 什么是数据库？

数据库（Database）是按数据结构组织、存储和管理数据的仓库。关系型数据库管理系统（RDBMS）以表格形式存储数据：

- 数据以表格形式出现
- 每行是一条记录
- 每列是一个字段（数据域）
- 多张表组成一个数据库

## RDBMS 常用术语

| 术语 | 说明 |
|------|------|
| 数据库 | 关联表的集合 |
| 数据表 | 数据矩阵，类似电子表格 |
| 列 | 同一类数据，如邮编 |
| 行 | 一组相关数据（记录） |
| 主键 | 唯一标识一行，一表通常一个主键 |
| 外键 | 关联两张表 |
| 索引 | 加速查询的结构，类似目录 |
| 参照完整性 | 不允许引用不存在的实体 |

## MySQL 特点

- 开源，支持标准 SQL
- 可处理千万级记录
- 支持多系统、多语言（C/C++/Java/Python/PHP 等）
- 表引擎常用 InnoDB（支持事务）
