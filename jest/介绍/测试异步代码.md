# 测试异步代码

参考文档: [测试异步代码](https://jest.nodejs.cn/docs/asynchronous)

对应实现见 `jest/async.test.js`。

## Promise

返回 Promise，Jest 会等其 resolve；若 reject，测试失败。

```js
test('promise', () => {
  return fetchData().then((data) => {
    expect(data).toBe('peanut butter');
  });
});
```

## async / await

给测试回调加 `async`，用 `await` 等待异步结果。

```js
test('async/await', async () => {
  const data = await fetchData();
  expect(data).toBe('peanut butter');
});
```

## 回调

若不用 Promise，用回调时必须调用 `done`，否则 Jest 不知道何时结束。

```js
test('callback', (done) => {
  function callback(error, data) {
    if (error) {
      done(error);
      return;
    }
    try {
      expect(data).toBe('peanut butter');
      done();
    } catch (error) {
      done(error);
    }
  }
  fetchData(callback);
});
```

## resolves / rejects

对 Promise 结果直接断言：

```js
await expect(fetchData()).resolves.toBe('peanut butter');
await expect(fetchData()).rejects.toMatch('error');
```
