# 模拟函数

参考文档: [模拟函数](https://jest.nodejs.cn/docs/mock-functions)

对应实现见 `jest/mock-functions.test.js`。

## 创建与用途

`jest.fn()` 创建模拟函数，可记录调用次数、参数、返回值，并自定义实现。

```js
const mockFn = jest.fn();
mockFn('a');
expect(mockFn).toHaveBeenCalled();
expect(mockFn).toHaveBeenCalledWith('a');
```

## 常用 API

* `mockReturnValue(value)` / `mockReturnValueOnce(value)`
* `mockResolvedValue` / `mockRejectedValue`：异步返回
* `mockImplementation(fn)` / `mockImplementationOnce(fn)`
* `mockClear()`：清调用记录；`mockReset()`：清记录并重置实现；`mockRestore()`：还原原实现（需 `jest.spyOn`）

## 断言匹配器

* `toHaveBeenCalled` / `toHaveBeenCalledTimes(n)`
* `toHaveBeenCalledWith(...args)` / `toHaveBeenLastCalledWith`
* `toHaveReturnedWith` / `toHaveReturnedTimes`

## 模拟模块

```js
jest.mock('./moduleName');
```

把模块里的导出换成模拟。也可配合 `jest.spyOn(obj, 'method')` 只监视某个方法。

更完整的模块级模拟见指南：手动模拟、ES6 类模拟。
