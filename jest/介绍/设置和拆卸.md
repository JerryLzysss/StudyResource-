# 设置和拆卸

参考文档: [设置和拆卸](https://jest.nodejs.cn/docs/setup-teardown)

对应实现见 `jest/setup-teardown.test.js`。

## 常用钩子

| 钩子 | 时机 |
|------|------|
| `beforeEach` | 每个测试前 |
| `afterEach` | 每个测试后 |
| `beforeAll` | 本文件/本 describe 全部测试前（一次） |
| `afterAll` | 本文件/本 describe 全部测试后（一次） |

适合初始化/清理共享状态，避免测试互相污染。

## 作用域

钩子写在 `describe` 内时，只作用于该块内的测试。

```js
describe('外层', () => {
  beforeEach(() => { /* 外层+内层测试都会跑 */ });

  describe('内层', () => {
    beforeEach(() => { /* 仅内层 */ });
    test('...', () => {});
  });
});
```

执行顺序：由外到内跑 `before*`，测试后再由内到外跑 `after*`。

## 只跑 / 跳过

* `test.only` / `describe.only`：只跑标记项（调试用）
* `test.skip` / `describe.skip`：跳过
* 也可临时用 `xit` / `xdescribe`、`fit` / `fdescribe`（视风格而定）
