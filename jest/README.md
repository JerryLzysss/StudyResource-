# Jest 学习笔记

参考文档: [Jest 中文网](https://jest.nodejs.cn/)

作用: JS 测试框架，配置简单，API 完善。

## 特点

- 零配置：开箱即用
- 快照：可与测试一起存在，也可内联
- 隔离：并行测试，提高性能
- API 齐全：工具包与文档集中
- 代码覆盖率：`--coverage` 查看覆盖率
- 模拟：自定义解析器与模拟对象
- 异常查明：失败时精确定位原因
