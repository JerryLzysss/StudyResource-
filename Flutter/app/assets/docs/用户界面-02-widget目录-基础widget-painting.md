These widgets apply visual effects to the children without changing their layout, size, or position.

                [BackdropFilter

                      A widget that applies a filter to the existing painted content and then paints a child. This effect is relatively expensive, especially if the filter...](https://api.flutter-io.cn/flutter/widgets/BackdropFilter-class.html)
                [ClipOval

A widget that clips its child using an oval.](https://api.flutter-io.cn/flutter/widgets/ClipOval-class.html)
                [ClipPath

A widget that clips its child using a path.](https://api.flutter-io.cn/flutter/widgets/ClipPath-class.html)
                [ClipRect

A widget that clips its child using a rectangle.](https://api.flutter-io.cn/flutter/widgets/ClipRect-class.html)
                [CustomPaint

A widget that provides a canvas on which to draw during the paint phase.](https://api.flutter-io.cn/flutter/widgets/CustomPaint-class.html)
                [DecoratedBox

A widget that paints a Decoration either before or after its child paints.](https://api.flutter-io.cn/flutter/widgets/DecoratedBox-class.html)
                [FractionalTranslation

                      A widget that applies a translation expressed as a fraction of the box's size before painting its child.](https://api.flutter-io.cn/flutter/widgets/FractionalTranslation-class.html)
                [Opacity

A widget that makes its child partially transparent.](https://api.flutter-io.cn/flutter/widgets/Opacity-class.html)
                [RotatedBox

A widget that rotates its child by a integral number of quarter turns.](https://api.flutter-io.cn/flutter/widgets/RotatedBox-class.html)
                [Transform

A widget that applies a transformation before painting its child.](https://api.flutter-io.cn/flutter/widgets/Transform-class.html)

更多 widget 请查看 [widget 目录](https://docs.flutter.cn/ui/widgets)。
