# 为什么使用markdown
Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。简洁、高效、易读、易写。同时，markdown也被Github等网站大量使用。
